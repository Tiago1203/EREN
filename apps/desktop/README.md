# @eren/desktop

The **desktop client** for EREN.

> **Status:** placeholder. No desktop framework (Electron, Tauri, …) has been
> chosen or wired yet. This directory reserves the architectural slot.

## Intended purpose

A native desktop surface for EREN, for workflows that benefit from offline
capability, local integrations, or a dedicated app experience. Like every app
it will stay thin, composing capability from `core/*` and `packages/*`.

## Layout

- `src/index.ts` — placeholder entry point.
- `package.json` — workspace manifest (no framework dependencies yet).
