# ADR-0600: Integration Architecture Overview

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

EPIC 6 connects EREN with external hospital systems, medical devices, and enterprise software. This enables data exchange and workflow automation.

## Decision

We will implement an **Adapter-based Integration Layer** with:

1. **Hospital Systems** — HL7, FHIR, DICOM
2. **Medical Devices** — Philips, GE, Dräger, Mindray
3. **Enterprise** — SAP, ServiceNow, Maximo, Azure AD

## Integration Patterns

| Pattern | Use Case |
|---------|----------|
| Request/Response | Synchronous queries |
| Pub/Sub | Event notifications |
| Batch | Bulk data sync |
| Streaming | Real-time telemetry |

## Consequences

### Positive
- Rich data integration
- Workflow automation
- Vendor flexibility

### Negative
- Complexity management
- Protocol overhead
