# ADR-0609: SAP Integration

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

SAP is used for ERP, inventory, and finance. EREN needs to sync equipment data and procurement.

## Decision

We will implement **SAP RFC/BAPI adapter**:

### Supported Operations

| Module | Operations | Use Case |
|--------|------------|----------|
| MM | Get/Create Purchase Order | Procurement |
| PM | Equipment Management | Asset sync |
| QM | Quality Management | Compliance |

### Implementation

```python
class SAPAdapter:
    async def connect(self, app_server: str, system: str):
        self.rfc = SAPRFC(app_server, system)
        await self.rfc.connect()
        
    async def get_equipment(self, equipment_id: str) -> Equipment:
        result = await self.rfc.call("EQUI_READ", {
            "EQUNR": equipment_id
        })
        return self.map_equipment(result)
```

## Consequences

### Positive
- ERP integration
- Financial tracking

### Negative
- Complex setup
- License costs
