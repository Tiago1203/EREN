# ADR-0602: HL7 V2 Integration

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

HL7 V2 remains widely used in hospitals for ADT messages, orders, and results.

## Decision

We will implement **HL7 V2.x message handling**:

### Supported Message Types

| Message | Trigger | Action |
|---------|---------|--------|
| ADT^A01 | Patient admission | Update context |
| ADT^A08 | Patient update | Sync info |
| MDM^T01 | Document creation | Log event |
| ORU^R01 | Result notification | Alert AI |

### Implementation

```python
class HL7Listener:
    async def on_message(self, hl7_message: str):
        parsed = HL7Parser.parse(hl7_message)
        await self.route_message(parsed)
        
    async def route_message(self, message: HL7Message):
        handlers = {
            "ADT": self.handle_adt,
            "MDM": self.handle_mdm,
            "ORU": self.handle_oru,
        }
        await handlers[message.type](message)
```

## Consequences

### Positive
- Legacy system support
- Real-time notifications

### Negative
- Complex parsing
- Protocol aging
