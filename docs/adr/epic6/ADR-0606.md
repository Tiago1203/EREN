# ADR-0606: Dräger Medical Integration

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Dräger provides anesthesia and ventilation equipment. EREN needs to monitor device status and maintenance schedules.

## Decision

We will implement **Dräger Infinity Gateway adapter**:

### Supported Devices

| Device | Protocol | Data |
|--------|----------|------|
| Evita V500 | MQTT | Ventilation |
| Fabius Tiro | RS232 | Anesthesia |
| Perseus | Network | Anesthesia |

### Implementation

```python
class DragerAdapter:
    async def connect_gateway(self, gateway_url: str):
        self.gateway = DragerGateway(gateway_url)
        await self.gateway.connect()
        
    async def get_device_status(self, device_id: str) -> DeviceStatus:
        return await self.gateway.get_status(device_id)
        
    async def subscribe_alerts(self, device_id: str):
        await self.gateway.subscribe_alerts(device_id, self.on_alert)
```

## Consequences

### Positive
- Ventilation monitoring
- Anesthesia integration

### Negative
- Legacy protocols
- Network complexity
