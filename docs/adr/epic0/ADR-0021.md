# ADR-0021: Evidence Required for CDS

**Status:** Accepted

**Date:** 2026-04-01

**Deciders:** Clinical Team

---

## Context

CDS recommendations without evidence are not clinically acceptable. EREN must provide citations for every recommendation to maintain clinical trust.

---

## Decision

**Every CDS recommendation must include supporting evidence with citations.**

No recommendation without evidence. Evidence must come from the Knowledge base, not be fabricated by the LLM.

---

## Evidence Requirements

| Element | Requirement |
|---------|-------------|
| Evidence source | Must cite a specific document in the Knowledge base |
| Citation format | Standard medical citation format (PMID, DOI, or internal doc ID) |
| Confidence score | 0.0-1.0, must reflect actual certainty |
| Reasoning chain | Must show the logical path from evidence to recommendation |
| Uncertainty | Must state what is unknown |

---

## Guardrail G5.1 (Epic 0.2)

> "CDS responses must include evidence citations for all claims"

If no evidence is found, EREN must say: "No evidence found in knowledge base for this query."
