# ADR-0061: CDS Evidence Requirements

**Status:** Accepted

**Date:** 2026-07-16

**Deciders:** Architecture Board

---

## Context

CDS recommendations without evidence are clinically unacceptable. EREN must provide verifiable evidence for every recommendation.

---

## Decision

**Every CDS recommendation must include at least one evidence citation from the Knowledge base.**

Evidence must:
- Reference a specific document (PMID, DOI, or internal doc ID)
- Include a relevance score
- Be retrieved from the Knowledge base (not fabricated)

If no evidence is found, EREN must explicitly state: "No evidence found in knowledge base for this query. Recommendation based on general medical knowledge."

---

## Evidence Retrieval

1. User query → semantic embedding
2. Qdrant search → top-k relevant documents
3. Filter by recency, source quality, relevance score
4. Include top evidence in CDS response
