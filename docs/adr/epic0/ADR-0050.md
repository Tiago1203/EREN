# ADR-0050: Hospital Context Architecture

**Status:** Accepted

**Date:** 2026-07-16

**Deciders:** Architecture Board

---

## Context

EREN serves hospital organizations with complex physical and organizational structures. The Hospital domain must model: Organization, Campus, Building, Floor, Room, Bed, Department.

---

## Decision

**The Hospital domain is modeled as a hierarchical containment structure:**

```
Organization
└── Campus (geographic location)
    └── Building
        └── Floor
            └── Room
                └── Bed
Department (cross-cuts hierarchy)
```

This structure enables capacity management, staffing optimization, and asset tracking.

See `EREN_BOUNDED_CONTEXT_MAP.md` (Epic 0 v1.1) for full context map.
