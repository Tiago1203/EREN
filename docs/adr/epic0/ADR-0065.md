**Status:** Proposed

**Date:** 2026-07-16

**Deciders:** Frontend Team

## Context

EREN needs a web interface for clinicians, biomedical engineers, and hospital administrators.

### Current Considerations

| Option | Pros | Cons |
|--------|------|------|
| Next.js (React) | SSR, good DX, team familiarity | Bundle size |
| SvelteKit | Smaller bundle, fast | Less ecosystem |
| Vue.js | Good documentation | Less common in enterprise |
