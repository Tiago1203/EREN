# ADR-0031: Prometheus + Grafana Observability

**Status:** Accepted

**Date:** 2026-05-01

**Deciders:** Infrastructure

---

## Context

EREN needs a complete observability stack: metrics, traces, and logs.

---

## Decision

**Prometheus + Grafana for metrics, Jaeger for traces, Loki for logs.** (Superseded by ADR-0085 for detailed implementation.)

---

## Summary

- Metrics: Prometheus + Grafana dashboards
- Tracing: Jaeger (OTLP)
- Logs: Loki + Grafana
- Alerting: Alertmanager + PagerDuty
