# ADR-0507: Calibration Advisor

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Medical devices require regular calibration to ensure accuracy. The Calibration Advisor helps schedule and execute calibration procedures.

## Decision

We will implement a **calibration management system**:

### Calibration Triggers
- Scheduled intervals
- Drift detection
- After repairs
- Critical measurements

### Calibration Records
- Device identification
- Calibration standards
- Environmental conditions
- Measurement results
- Next due date

### Auto-Scheduling
```
Next Calibration = Last Calibration + Interval - Lead Time
```

## Consequences

### Positive
- Ensures device accuracy
- Compliance documentation
- Proactive scheduling

### Negative
- Requires accurate intervals
- May interrupt operations
