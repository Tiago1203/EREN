# ADR-0705: Incident Management UI

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Incident Management UI provides tools for tracking and resolving device incidents.

## Decision

We will implement incident lifecycle management:

### Features

1. **Incident Board** — Kanban-style workflow
2. **Incident Detail** — Full investigation view
3. **Timeline** — Event history
4. **AI Analysis** — Root cause suggestions
5. **Collaboration** — Comments, mentions

### Workflow States

```
┌─────────┐    ┌───────────┐    ┌────────────┐    ┌──────────┐
│ New     │ → │ Investigating│ → │ Resolved  │ → │ Closed  │
└─────────┘    └───────────┘    └────────────┘    └──────────┘
```
