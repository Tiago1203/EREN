# ADR-0700: User Experience Architecture

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

EPIC 7 implements the User Experience layer for EREN вҖ” the interface that brings all backend services together.

## Dependencies

- **EPIC 0** вҖ” Philosophy, Architecture
- **EPIC 1** вҖ” Infrastructure Platform
- **EPIC 2** вҖ” Domain Models (Device, Incident)
- **EPIC 3** вҖ” Hospital Contexts
- **EPIC 4** вҖ” AI Core (Cognitive Runtime)
- **EPIC 5** вҖ” Clinical Intelligence (CDSS)
- **EPIC 6** вҖ” Integrations

## Decision

We will implement a **Multi-Platform User Experience**:
