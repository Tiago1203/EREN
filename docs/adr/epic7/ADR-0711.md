# ADR-0711: Internationalization

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

EREN supports multilingual users across different hospitals.

## Decision

We will implement **i18n with next-i18next**:

### Supported Languages

| Language | Code | Status |
|----------|------|--------|
| English | en | Primary |
| Spanish | es | Secondary |
| Portuguese | pt | Secondary |
| German | de | Future |

### Implementation

```typescript
// i18n.ts
export const languages = ["en", "es", "pt"] as const;
export type Language = (typeof languages)[number];

// Usage
const { t } = useTranslation();
return <h1>{t("dashboard.title")}</h1>;
```

### Translation Keys

```
common.save
common.cancel
common.error
dashboard.title
device.list
incident.create
```
