# ADR-0809: Backup Strategy

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Robust backup strategy ensures data can be recovered in case of failure.

## Decision

We will implement **3-2-1 Backup Strategy**:

### Backup Tiers

| Tier | Type | Frequency | Retention |
|------|------|-----------|-----------|
| Daily | Incremental | Daily | 7 days |
| Weekly | Full | Weekly | 4 weeks |
| Monthly | Full | Monthly | 12 months |
| Offsite | Full | Monthly | 7 years |

### Data Backed Up

```yaml
backup_targets:
  database:
    - PostgreSQL (full)
    - Redis (AOF)
  files:
    - /data/uploads (incremental)
    - /data/exports (full)
  configs:
    - Kubernetes configs
    - Secrets (encrypted)
```

### Verification

- Weekly restore tests
- Monthly DR drills
- Annual full recovery simulation
