# ADR-1011: Documentation Standards

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Estándares de documentación.

---

## Decision

Documentation standards:

1. **User Documentation**
   - Getting started
   - User guides
   - How-to articles

2. **Admin Documentation**
   - Installation guide
   - Configuration reference
   - Operations guide

3. **Developer Documentation**
   - API reference
   - Architecture
   - Contributing guide

---

## Consequences

**Pros:**
- Usability
- Adoption
- Support reduction

**Cons:**
- Maintenance effort
- Translation costs
