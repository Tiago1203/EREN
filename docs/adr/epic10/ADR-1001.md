# ADR-1001: Packaging Strategy

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

EREN necesita un estrategia de empaquetado para distribución empresarial.

---

## Decision

Estrategia de empaquetado:

1. **Docker Images**
   - Multi-stage builds
   - Distroless base
   - Vulnerability scanning

2. **Helm Charts**
   - Versioned charts
   - OCI registry
   - Signed packages

3. **Installers**
   - CLI installer
   - Docker Compose for dev
   - K8s manifests

---

## Consequences

**Pros:**
- Flexibilidad de deployment
- Seguridad en imágenes
- Versionado claro

**Cons:**
- Múltiples artefactos
- Testing adicional
