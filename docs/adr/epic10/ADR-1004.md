# ADR-1004: Licensing Architecture

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Sistema de licenciamiento para producto comercial.

---

## Decision

Licensing con:

1. **License Server**
   - JWT-based licenses
   - License validation
   - Feature flags

2. **License Types**
   - Community
   - Professional
   - Enterprise

3. **Tracking**
   - Usage metrics
   - Expiration alerts
   - Compliance reports

---

## Consequences

**Pros:**
- Revenue protection
- Feature control
- Compliance

**Cons:**
- Complexity
- License server dependency
