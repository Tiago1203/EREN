# ADR-1009: Backup & Recovery Enterprise

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Backup y recovery para empresa.

---

## Decision

Enterprise backup:

1. **Backup Types**
   - Full backup daily
   - Incremental hourly
   - Point-in-time recovery

2. **Storage**
   - Local snapshots
   - Off-site backup
   - Encryption at rest

3. **Recovery**
   - RTO < 4 hours
   - RPO < 1 hour
   - Tested quarterly

---

## Consequences

**Pros:**
- Data protection
- Compliance
- Peace of mind

**Cons:**
- Storage costs
- Complexity
