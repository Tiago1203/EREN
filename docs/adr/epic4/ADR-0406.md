# ADR-0406: Confidence Scoring Model

**Status:** Proposed

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

EREN debe expresar confianza en sus respuestas. Esto es crítico para:
1. **Seguridad clínica** — Si EREN no está seguro, debe decirlo
2. **Expectativas del usuario** — El usuario sabe qué esperar
3. **Escalamiento** — Baja confianza → escalar a profesional

Epic 0 establece: "EREN always measures and expresses uncertainty."

---

## Decision

**Confidence Model:**

```python
@dataclass
class ConfidenceScore:
    """Confidence score con breakdown."""
    
    # Overall score 0.0 - 1.0
    overall: float
    
    # Component scores
    evidence_score: float    # How much evidence supports
    consistency_score: float # Do sources agree?
    coverage_score: float    # Did we cover all aspects?
    recency_score: float     # How recent is the information?
    
    # Warnings
    low_evidence: bool
    conflicting_sources: bool
    outdated_information: bool
    
    # Human-readable summary
    summary: str
    
    @property
    def level(self) -> ConfidenceLevel:
        if self.overall >= 0.9:
            return ConfidenceLevel.HIGH
        elif self.overall >= 0.7:
            return ConfidenceLevel.MEDIUM
        elif self.overall >= 0.5:
            return ConfidenceLevel.LOW
        else:
            return ConfidenceLevel.VERY_LOW

class ConfidenceLevel(Enum):
    HIGH = "high"         # 0.9-1.0
    MEDIUM = "medium"     # 0.7-0.9
    LOW = "low"           # 0.5-0.7
    VERY_LOW = "very_low" # 0.0-0.5
```

**Calculation Algorithm:**

```python
async def calculate_confidence(
    self,
    reasoning_trace: ReasoningResult,
    evidence: list[Evidence],
    context: ReasoningContext,
) -> ConfidenceScore:
    """Calculate multi-dimensional confidence."""
    
    # 1. Evidence score
    evidence_score = self.calculate_evidence_score(evidence)
    
    # 2. Consistency score (do sources agree?)
    consistency_score = self.calculate_consistency_score(evidence)
    
    # 3. Coverage score (did we cover all aspects?)
    coverage_score = self.calculate_coverage_score(
        reasoning_trace,
        context.query,
    )
    
    # 4. Recency score (how recent is information?)
    recency_score = self.calculate_recency_score(evidence)
    
    # Weighted combination
    overall = (
        0.40 * evidence_score +
        0.25 * consistency_score +
        0.20 * coverage_score +
        0.15 * recency_score
    )
    
    # Calibration (optional: use temperature scaling)
    overall = self.calibrate(overall)
    
    # Warnings
    warnings = self.generate_warnings(
        evidence_score,
        consistency_score,
        coverage_score,
        recency_score,
    )
    
    return ConfidenceScore(
        overall=overall,
        evidence_score=evidence_score,
        consistency_score=consistency_score,
        coverage_score=coverage_score,
        recency_score=recency_score,
        **warnings,
        summary=self.generate_summary(overall, warnings),
    )

async def calculate_evidence_score(
    self,
    evidence: list[Evidence],
) -> float:
    """Score based on evidence quantity and quality."""
    if not evidence:
        return 0.0
    
    # Quantity
    evidence_count_score = min(len(evidence) / 5, 1.0)
    
    # Quality (source authority)
    authority_scores = [e.source.authority_score for e in evidence]
    quality_score = sum(authority_scores) / len(authority_scores)
    
    # Relevance
    relevance_scores = [e.relevance_score for e in evidence]
    relevance_score = sum(relevance_scores) / len(relevance_scores)
    
    return 0.4 * evidence_count_score + 0.3 * quality_score + 0.3 * relevance_score

async def calculate_consistency_score(
    self,
    evidence: list[Evidence],
) -> float:
    """Score based on agreement between sources."""
    if len(evidence) < 2:
        return 1.0  # No conflict possible
    
    # Compare claims
    claims = [e.claim for e in evidence if e.claim]
    
    if len(claims) < 2:
        return 1.0
    
    # Count agreements
    agreements = 0
    for i, c1 in enumerate(claims):
        for c2 in claims[i+1:]:
            if self.semantic_similarity(c1, c2) > 0.85:
                agreements += 1
    
    total_pairs = len(claims) * (len(claims) - 1) / 2
    return agreements / total_pairs if total_pairs > 0 else 1.0
```

**Response Formatting:**

```python
async def format_confidence(
    self,
    confidence: ConfidenceScore,
) -> str:
    """Format confidence for user display."""
    level = confidence.level
    
    if level == ConfidenceLevel.HIGH:
        return ""  # No disclaimer needed
    
    elif level == ConfidenceLevel.MEDIUM:
        return (
            f"Estoy razonablemente seguro de esta respuesta (confianza: {confidence.overall:.0%}). "
            f"{confidence.summary}"
        )
    
    elif level == ConfidenceLevel.LOW:
        return (
            f"Esta respuesta tiene incertidumbre moderada (confianza: {confidence.overall:.0%}). "
            f"Te recomiendo verificar con fuentes adicionales. {confidence.summary}"
        )
    
    else:  # VERY_LOW
        return (
            f"⚠️ No tengo suficiente confianza en esta respuesta (confianza: {confidence.overall:.0%}). "
            f"Por favor consulta con un profesional antes de actuar. {confidence.summary}"
        )
```

## Action Thresholds

| Confidence | Level | User Experience | Action |
|------------|-------|-----------------|--------|
| 0.90-1.00 | HIGH | Normal response | None |
| 0.70-0.89 | MEDIUM | Response + disclaimer | Optional escalation |
| 0.50-0.69 | LOW | Response + warning | Suggest escalation |
| 0.00-0.49 | VERY_LOW | Warning only | Require acknowledgment |

---

## Reasons

**1. Safety:**
- Users know when to trust EREN
- Escalation prevents dangerous decisions
- Transparency builds trust

**2. Calibrated uncertainty:**
- Not overconfident when uncertain
- Not underconfident when sure
- Learns from feedback

**3. Multi-dimensional:**
- Different failure modes detected
- Actionable feedback
- Improvement tracking

---

## Consequences

**Positive:**
- Safer AI
- Transparent uncertainty
- User trust
- Improvement feedback loop

**Negative:**
- May seem less confident than capable
- Calibration requires tuning
- More complex response generation
