# ADR-0409: Memory Architecture

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

EREN necesita memoria para:
1. **Conversation context** — Historial de la conversación actual
2. **Session state** — Estado entre mensajes
3. **Long-term knowledge** — Lo aprendido persiste
4. **Privacy** — Memorias sensibles deben ser protegidas

Epic 0 Cognitive Model define Remember/Learn. Este ADR define la arquitectura de memoria.

---

## Decision

**Memory Architecture - 3 Tier System:**

```
┌──────────────────────────────────────────────────────────────┐
│                    Memory Manager                            │
│                                                              │
│  ┌──────────────────────────────────────────────────────┐   │
│  │              Unified Memory Interface                  │   │
│  │                                                       │   │
│  │  store(key, value, type, ttl)                         │   │
│  │  retrieve(key, type) -> value                        │   │
│  │  forget(key)                                          │   │
│  │  consolidate()                                        │   │
│  └──────────────────────────────────────────────────────┘   │
└──────────────────────────────────────────────────────────────┘
         │                   │                   │
         ▼                   ▼                   ▼
┌─────────────────┐ ┌─────────────────┐ ┌─────────────────┐
│ Working Memory  │ │ Session Memory  │ │ Long-term Memory│
│ (Redis)         │ │ (Redis)         │ │ (PostgreSQL)    │
│                 │ │                 │ │                 │
│ • Context       │ │ • Conversation  │ │ • Learned facts │
│ • Temp data     │ │   history      │ │ • User prefs    │
│ • RAG cache     │ │ • State        │ │ • Domain KB     │
│                 │ │                 │ │ • Entity cache │
│ TTL: 5 min      │ │ TTL: 30 min    │ │ TTL: permanent │
└─────────────────┘ └─────────────────┘ └─────────────────┘
```

**Memory Types:**

```python
class MemoryType(Enum):
    WORKING = "working"      # Transient context
    SESSION = "session"      # Conversation context
    LONG_TERM = "long_term"  # Persistent knowledge
    EPISODIC = "episodic"    # Event memories
    SEMANTIC = "semantic"    # Facts and knowledge

@dataclass
class MemoryBlock:
    id: MemoryId
    memory_type: MemoryType
    key: str
    value: Any
    metadata: dict
    
    # Provenance
    created_at: datetime
    created_by: str  # user_id or system
    source: str  # conversation_id, tool_name, etc.
    
    # Utility
    importance: float  # 0.0-1.0
    access_count: int
    last_accessed: datetime
    expires_at: datetime | None
    
    # Privacy
    is_sensitive: bool
    retention_policy: RetentionPolicy
```

**Working Memory:**

```python
class WorkingMemory:
    """Transient memory for current processing."""
    
    def __init__(self, redis: Redis):
        self.redis = redis
        self.ttl = 300  # 5 minutes
    
    async def store(
        self,
        key: str,
        value: Any,
        metadata: dict | None = None,
    ) -> MemoryBlock:
        """Store in working memory."""
        block = MemoryBlock(
            id=MemoryId.generate(),
            memory_type=MemoryType.WORKING,
            key=key,
            value=value,
            metadata=metadata or {},
            created_at=datetime.utcnow(),
            created_by="system",
            source="working_memory",
            importance=0.5,
            access_count=0,
            last_accessed=datetime.utcnow(),
            expires_at=datetime.utcnow() + timedelta(seconds=self.ttl),
            is_sensitive=False,
            retention_policy=RetentionPolicy.EPHEMERAL,
        )
        
        await self.redis.setex(
            f"working:{key}",
            self.ttl,
            json.dumps(block.__dict__),
        )
        
        return block
    
    async def retrieve(self, key: str) -> MemoryBlock | None:
        """Retrieve from working memory."""
        data = await self.redis.get(f"working:{key}")
        if not data:
            return None
        
        block = MemoryBlock(**json.loads(data))
        block.access_count += 1
        block.last_accessed = datetime.utcnow()
        
        return block
```

**Session Memory:**

```python
class SessionMemory:
    """Conversation-scoped memory."""
    
    def __init__(self, redis: Redis):
        self.redis = redis
        self.ttl = 1800  # 30 minutes
        self.max_history = 50  # Messages in context window
    
    async def add_message(
        self,
        conversation_id: ConversationId,
        message: Message,
    ) -> None:
        """Add message to conversation history."""
        key = f"session:{conversation_id}:history"
        
        # Add to list
        await self.redis.rpush(key, json.dumps(message.__dict__))
        
        # Trim to max size
        await self.redis.ltrim(key, -self.max_history, -1)
        
        # Refresh TTL
        await self.redis.expire(key, self.ttl)
    
    async def get_history(
        self,
        conversation_id: ConversationId,
        limit: int = 10,
    ) -> list[Message]:
        """Get recent conversation history."""
        key = f"session:{conversation_id}:history"
        
        # Get last N messages
        messages_data = await self.redis.lrange(key, -limit, -1)
        
        return [Message(**json.loads(m)) for m in messages_data]
    
    async def store_preference(
        self,
        conversation_id: ConversationId,
        preference: UserPreference,
    ) -> None:
        """Store user preference for session."""
        key = f"session:{conversation_id}:preferences"
        await self.redis.hset(key, preference.name, json.dumps(preference.value))
        await self.redis.expire(key, self.ttl)
```

**Long-term Memory:**

```python
class LongTermMemory:
    """Persistent memory stored in PostgreSQL."""
    
    def __init__(self, db: Database):
        self.db = db
    
    async def store(
        self,
        memory_type: MemoryType,
        key: str,
        value: Any,
        metadata: dict,
        retention: RetentionPolicy,
    ) -> MemoryId:
        """Store in long-term memory."""
        memory_id = MemoryId.generate()
        
        await self.db.execute("""
            INSERT INTO memories (id, type, key, value, metadata, retention, created_at)
            VALUES ($1, $2, $3, $4, $5, $6, NOW())
        """, memory_id, memory_type.value, key, json.dumps(value), metadata, retention.value)
        
        return memory_id
    
    async def retrieve(
        self,
        memory_type: MemoryType,
        key: str,
    ) -> list[MemoryBlock]:
        """Retrieve from long-term memory."""
        rows = await self.db.query("""
            SELECT * FROM memories 
            WHERE type = $1 AND key = $2
            ORDER BY created_at DESC
            LIMIT 100
        """, memory_type.value, key)
        
        return [MemoryBlock(**row) for row in rows]
    
    async def consolidate(self) -> int:
        """
        Consolidate important session memories to long-term.
        Called periodically (e.g., hourly).
        """
        # Find high-importance recent memories
        result = await self.db.execute("""
            SELECT * FROM memories 
            WHERE type = 'session'
            AND importance > 0.7
            AND created_at > NOW() - INTERVAL '1 hour'
        """)
        
        consolidated = 0
        for row in result:
            # Promote to semantic memory
            await self.store(
                MemoryType.SEMANTIC,
                row['key'],
                row['value'],
                {**row['metadata'], 'consolidated_from': row['id']},
                RetentionPolicy.LONG_TERM,
            )
            consolidated += 1
        
        return consolidated
```

**Memory Manager:**

```python
class MemoryManager:
    """Unified interface to all memory types."""
    
    def __init__(
        self,
        working: WorkingMemory,
        session: SessionMemory,
        long_term: LongTermMemory,
    ):
        self.working = working
        self.session = session
        self.long_term = long_term
    
    async def store(
        self,
        key: str,
        value: Any,
        memory_type: MemoryType,
        ttl: int | None = None,
        importance: float = 0.5,
        is_sensitive: bool = False,
    ) -> MemoryId:
        """Store memory in appropriate tier."""
        if memory_type == MemoryType.WORKING:
            return await self.working.store(key, value)
        
        elif memory_type == MemoryType.SESSION:
            return await self.session.store(key, value)
        
        elif memory_type == MemoryType.LONG_TERM:
            retention = RetentionPolicy.LONG_TERM if not is_sensitive else RetentionPolicy.SENSITIVE
            return await self.long_term.store(memory_type, key, value, {}, retention)
        
        else:
            raise ValueError(f"Unknown memory type: {memory_type}")
    
    async def retrieve(self, key: str) -> list[MemoryBlock]:
        """Retrieve from all tiers."""
        results = []
        
        # Check in order: working > session > long_term
        if w := await self.working.retrieve(key):
            results.append(w)
        
        if s := await self.session.retrieve(key):
            results.append(s)
        
        if l := await self.long_term.retrieve(MemoryType.SEMANTIC, key):
            results.extend(l)
        
        return results
    
    async def forget(self, key: str, memory_type: MemoryType | None = None) -> None:
        """Remove memory."""
        if memory_type == MemoryType.WORKING or memory_type is None:
            await self.working.forget(key)
        if memory_type == MemoryType.SESSION or memory_type is None:
            await self.session.forget(key)
        if memory_type == MemoryType.LONG_TERM or memory_type is None:
            await self.long_term.forget(key)
```

## Privacy Considerations

| Memory Type | Sensitivity | Retention | Access Control |
|-------------|-------------|-----------|----------------|
| Working | Low | 5 min | System only |
| Session | Medium | 30 min | User + System |
| Long-term | Variable | Configurable | RBAC |
| Episodic | High | Until consolidated | User only |
| Semantic | Low | Permanent | System + Analytics |

---

## Reasons

**1. Performance:**
- Working memory (Redis) for speed
- Session memory for conversation continuity
- Long-term for persistence

**2. Privacy:**
- Sensitive data gets special handling
- Retention policies per type
- User control over memories

**3. Scalability:**
- Redis handles hot data
- PostgreSQL handles cold data
- Consolidation prevents memory bloat

---

## Consequences

**Positive:**
- Fast context retrieval
- Persistent learning
- Privacy-aware
- Scalable architecture

**Negative:**
- Redis dependency
- More complex than single-store
- Consolidation adds overhead
