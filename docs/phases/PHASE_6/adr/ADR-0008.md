# ADR-0008: Administration & Connectors Architecture

*Status: Proposed*
*Date: 2026-07-24*
*Deciders: Architecture Board*

---

## Context

Necesitamos implementar el panel administrativo final que:
- Gestione usuarios, roles y permisos
- Configure opciones del sistema
- Prepare arquitectura para conectores FHIR, HL7, DICOM
- Registre auditoría del sistema

---

## Decision

### Arquitectura de Módulos

1. **Admin Module**: Gestión de usuarios, roles, configuraciones
2. **Connectors Module**: Framework para integraciones externas
3. **Stores dedicados**: Zustand para estado admin
4. **Connector Registry**: Patrón Registry para conectores

### Estructura

```
modules/
├── admin/                    # Gestión administrativa
│   ├── components/          # UserList, RoleManager, etc.
│   ├── services/            # AdminService
│   └── types/               # User, Role, Permission
│
└── connectors/              # Framework de conectores
    ├── registry/            # ConnectorRegistry
    └── adapters/            # FHIR, HL7, DICOM
```

### Connector Registry Pattern

```typescript
// Patrón Registry para conectores
class ConnectorRegistry {
  private connectors: Map<string, Connector>;
  
  register(connector: Connector): void;
  get(id: string): Connector | undefined;
  list(): Connector[];
}
```

---

## Consequences

### Positive
- Gestión centralizada de usuarios
- Permisos granulares
- Auditoría completa
- Preparado para integrations

### Negative
- Complejidad de permisos
- Sincronización de estado

### Neutral
- Conectores no implementados aún (preparados para FASE 7)

---

## Alternatives Considered

### Alternative 1: RBAC simple
- **Rechazado**: No escala bien

### Alternative 2: ABAC (Attribute-Based)
- **Aceptado para futuro**: Más flexible pero complejo

### Auditoría Automática

```typescript
// Interceptor para auditoría
function auditLog(action: string, resource: string) {
  return function(target: any, propertyKey: string, descriptor: PropertyDescriptor) {
    const original = descriptor.value;
    descriptor.value = async function(...args) {
      const result = await original.apply(this, args);
      await auditService.log({ action, resource, ... });
      return result;
    };
  };
}
```
