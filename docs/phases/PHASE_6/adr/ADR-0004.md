# ADR-0004: AI Center Architecture

*Status: Proposed*
*Date: 2026-07-24*
*Deciders: Architecture Board*

---

## Context

Necesitamos implementar el Centro de IA que integra:
- Chat conversacional con AI Kernel (PHASE 2)
- Agentes especializados (PHASE 5)
- Contexto clínico (EPIC 12)
- Manejo de incertidumbre (EPIC 14)

Este módulo es crítico porque conecta el frontend con toda la inteligencia artificial del sistema.

---

## Decision

### Arquitectura del Módulo AI

1. **Cliente AI SDK**: Abstracción sobre AI Kernel
2. **Stores locales**: Chat y agentes en Zustand
3. **Servicios dedicados**: ChatService, AgentService, ContextService
4. **Componentes modulares**: ChatInterface, AgentPanel, ContextBuilder

### Flujo de Chat

```
User Input → ChatInterface → ChatService → AI Kernel (PHASE 2)
                                              ↓
                                         AI Agents (PHASE 5)
                                              ↓
                                         Clinical Context
                                              ↓
                                         Response ← ChatMessage
```

### Integración PHASE 2/5

```typescript
// packages/ai-sdk/src/client.ts
export interface AIKernelClient {
  chat(request: ChatRequest): Promise<ChatResponse>;
  getAgents(): Promise<Agent[]>;
  getConversations(): Promise<Conversation[]>;
}
```

---

## Consequences

### Positive
- Integración limpia con AI Kernel
- Agentes especializados accesibles
- Contexto clínico configurable
- Historial de conversaciones

### Negative
- Dependiente de PHASE 2 y PHASE 5
- Requiere API de backend

### Neutral
- Manejo de streaming para UX
- Retry logic para resilience

---

## Alternatives Considered

### Alternative 1: Llamadas directas a API
- **Rechazado**: Acoplamiento directo, difícil de testear

### Alternative 2: AI SDK Package (Elegido)
- **Aceptado**: Abstracción limpia, reusable

## Implementation Notes

### Chat Flow

```typescript
// 1. User sends message
const message = await chatService.sendMessage(
  input,
  context,
  selectedAgentId
);

// 2. Service calls AI Kernel

// 3. Response includes uncertainty
if (response.uncertainty > 0.5) {
  showUncertaintyIndicator(response.uncertainty);
}
```
