# ADR-0071: Cognitive Memory Enhancement

*Status: Proposed*
*Date: 2026-07-24*
*Deciders: Architecture Board*

---

## Context

La auditoría arquitectónica identificó que el modelo de memoria actual es parcial, sin modelo cognitivo completo. Los agentes no tienen memoria episódica, semántica y procedimental.

**Problema:**
- Solo memoria episódica básica
- Sin memoria semántica
- Sin memoria procedimental
- Sin consolidación de memoria

---

## Decision

Actualizar **EPIC 9 (Agent Memory Engine)** a versión 2.0 con Cognitive Memory Model:

1. **SemanticMemory**: Memoria de conocimiento general
2. **ProceduralMemory**: Memoria de procedimientos y habilidades
3. **WorkingMemory**: Memoria de trabajo activa
4. **MemoryConsolidation**: Consolidación de episódica a largo plazo

---

## Consequences

### Positive
- Memoria cognitiva completa
- Aprendizaje de experiencias
- Conocimiento compartible
- Base para learning continuo

### Negative
- Complejidad en gestión de memorias
- Storage requirements

### Neutral
- Dependiente de FASE 2 (Memory)
- Necesita integración con EPIC 10

---

## Alternatives Considered

### Alternative 1: Memoria simple
- Solo episódica
- **Rechazado**: Insuficiente para CDSS

### Alternative 2: Triple Store Memory (Elegido)
- Episódica + Semántica + Procedimental
- **Aceptado**: Modelo cognitivo completo

### Alternative 3: Vector Memory
- Solo embeddings
- **Rechazado**: No estructurado

---

class CognitiveMemory:
