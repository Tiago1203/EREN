## 2. ADRs - Rutas Anticuradas

### ⚠️ Problema Crítico

Todos los ADRs y README de EPICs usan rutas anticuadas:

```
❌ from core.PHASE_2.ai.foundation import AIKernel
❌ from core.PHASE_2.ai.domain import DeviceGateway
❌ from core.PHASE_2.ai.prompt import ...
```

Deberían usar:
```
✅ from core.PHASE_2.ai.foundation import AIKernel
✅ from core.PHASE_2.ai.domain import DeviceGateway
✅ from core.PHASE_2.ai.prompt import ...
```
