# ADR-2124: Memory & Learning System

## Estado

**Aceptado** - 2026-07-20

## Contexto

Epic 15 implementa el Memory System — el sistema que mantiene contexto, aprende de interacciones, y mejora continuamente.

El Memory System es responsable de:
- Mantener contexto de conversación
- Almacenar conocimiento a largo plazo
- Recordar episodios pasados
- Aprender de feedback
- Mejorar con cada interacción

## Decisiones

### Tipos de Memoria

| Tipo | Descripción | TTL |
|------|-------------|-----|
| Working | Contexto inmediato | Sesión |
| Short-term | Memoria reciente | Horas |
| Long-term | Conocimiento persistente | Indefinido |
| Episodic | Experiencias pasadas | Indefinido |

## Consecuencias

### Positivas
- Contexto mantenido entre sesiones
- Aprendizaje continuo
- Memoria institucional persistente

### Negativas
- Almacenamiento creciente
- Gestión de olvidos selectivos
