# ADR-2900: AI Core Architecture

## Estado
**Accepted** ✅

## Contexto
Necesitamos integrar todos los componentes del AI Core.

## Decisión
Implementar AICoreController:

```python
core/ai/integration/
├── __init__.py       # Exports
├── models.py         # AICoreConfig, AICoreStats
└── controller.py    # AICoreController
```

### Flujo de Integración
```
User Input
    │
    ▼
┌───────────────────────┐
│ Session Manager       │
│ (get/create session)  │
└───────────────────────┘
    │
    ▼
┌───────────────────────┐
│ Context Builder       │
│ (build context)       │
└───────────────────────┘
    │
    ▼
┌───────────────────────┐
│ Prompt Builder       │
│ (render prompt)      │
└───────────────────────┘
    │
    ▼
┌───────────────────────┐
│ Provider Manager     │
│ (call LLM)           │
└───────────────────────┘
    │
    ▼
┌───────────────────────┐
│ Response Composer     │
│ (format response)    │
└───────────────────────┘
    │
    ▼
Response
```

## Componentes Conectados
| Componente | Función |
|-----------|---------|
| ConversationController | Gestión de conversaciones |
| MemoryManager | Sistema de memoria |
| PromptBuilder | Construcción de prompts |
| ContextBuilder | Construcción de contexto |
| ToolOrchestrator | Orquestación de herramientas |
| ProviderManager | Gestión de proveedores |
| ResponseComposer | Composición de respuestas |
| SessionManager | Gestión de sesiones |

## Consecuencias
- Positivas: Arquitectura unificada
- Positivas: Fácil extensibilidad
- Neutrales: Complejidad de integración
