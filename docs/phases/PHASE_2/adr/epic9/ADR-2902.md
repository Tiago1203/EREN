# ADR-2902: AI Core Configuration

## Estado
**Accepted** ✅

## Contexto
El AI Core necesita configuración flexible.

## Decisión

### AICoreConfig
```python
@dataclass
class AICoreConfig:
    # Providers
    default_provider: str = "openai"
    fallback_providers: list[str] = []
    
    # Límites
    max_tokens: int = 4096
    max_context_tokens: int = 128000
    
    # Features
    enable_memory: bool = True
    enable_tools: bool = True
    enable_streaming: bool = True
    
    # Sesiones
    session_timeout_minutes: int = 30
    default_token_budget: int = 100000
    
    # Prompt
    default_strategy: str = "direct"
    default_temperature: float = 0.7
```

### AICoreStats
```python
@dataclass
class AICoreStats:
    status: AICoreStatus
    memory_initialized: bool
    tools_initialized: bool
    providers_initialized: bool
    metrics: AICoreMetrics
    uptime_seconds: float
```

### AICoreMetrics
```python
@dataclass
class AICoreMetrics:
    total_requests: int
    successful_requests: int
    failed_requests: int
    total_tokens_used: int
    total_cost: float
    avg_latency_ms: float
```

## Consecuencias
- Positivas: Configuración flexible
- Positivas: Métricas claras
- Neutrales: Complejidad de configuración
