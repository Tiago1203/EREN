# ADR-2000: AI Foundation Architecture

## Estado
**Accepted** ✅

## Contexto
Necesitamos establecer la arquitectura base del AI Core que será la fundación sobre la cual todas las demás capacidades de IA se construirán.

## Decisión
Implementar el AI Foundation con la siguiente estructura:

```
core/ai/
├── kernel/          # AIKernel - Núcleo central
├── contracts/       # Interfaces y abstracciones
├── dto/             # Data Transfer Objects
├── exceptions/      # Jerarquía de excepciones
├── config/          # Configuración
├── registry/        # Registro de modelos
├── providers/       # Abstracción de proveedores
├── di/             # Inyección de dependencias
├── context/        # Objetos de contexto
└── interfaces/      # Alias para contracts
```

## Consecuencias
- Positivas: Base sólida y extensible para todas las capacidades de IA
- Positivas: Separación clara de responsabilidades
- Negativas: Curva de aprendizaje inicial más alta
- Neutrales: Requiere configuración inicial

## Implementación
Ver `core/ai/` en el código fuente.
