# ADR-2001: Contract Design

## Estado
**Accepted** ✅

## Contexto
Necesitamos definir interfaces claras y abstracciones para los componentes del AI Core.

## Decisión
Implementar contratos como interfaces ABC con métodos abstractos:

- `AIProvider`: Contrato base para proveedores
- `ModelRegistry`: Registro de modelos
- `AIConfiguration`: Configuración
- `Container`: Inyección de dependencias
- `Tool`: Herramientas ejecutables
- `AIKernel`: Kernel principal

## Consecuencias
- Positivas: Contratos claros para implementación
- Positivas: Fácil de testear con mocks
- Neutrales: Requiere implementar todas las interfaces
