# ADR-2600: Response Architecture

## Estado
**Accepted** ✅

## Contexto
Necesitamos un sistema flexible de construcción de respuestas.

## Decisión
Implementar ResponseComposer:

```python
core/ai/response/
├── models.py        # Response, ResponseElement
└── composer.py     # ResponseComposer
```

Tipos de respuesta:
- TEXT, MARKDOWN, TABLE, CODE
- IMAGE, AUDIO, VIDEO, FILE
- JSON, HTML, MIXED

## Elementos Soportados
| Tipo | Descripción |
|------|-------------|
| TEXT | Texto plano |
| MARKDOWN | Contenido Markdown |
| CODE | Bloque de código |
| TABLE | Tabla estructurada |
| IMAGE | Imagen |
| REFERENCE | Referencia/Cita |
| LIST | Lista |
| WARNING/INFO/ERROR | Notificaciones |

## Consecuencias
- Positivas: Flexibilidad en respuestas
- Positivas: Composición modular
- Neutrales: Curva de aprendizaje
