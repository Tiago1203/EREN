# ADR-0608: ServiceNow Integration

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

ServiceNow is widely used for IT service management. EREN needs to create tickets and sync work orders.

## Decision

We will implement **ServiceNow adapter** using REST API:

### Supported Operations

| Operation | Use Case |
|-----------|----------|
| Create Incident | Alert ticket |
| Create Task | Work order |
| Update Record | Status sync |
| Attachment | Upload reports |

### Implementation

```python
class ServiceNowAdapter:
    async def create_incident(
        self,
        title: str,
        description: str,
        priority: str,
    ) -> str:
        incident = await self.api.create("incident", {
            "short_description": title,
            "description": description,
            "priority": priority,
        })
        return incident.sys_id
    
    async def attach_document(self, record_id: str, doc: bytes):
        await self.api.attach("incident", record_id, doc)
```

## Consequences

### Positive
- ITSM integration
- Workflow automation

### Negative
- API rate limits
- Table configuration
