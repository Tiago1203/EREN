# ADR-2601: Response Elements

## Estado
**Accepted** ✅

## Contexto
Cada elemento de respuesta necesita formato específico.

## Decisión

### ResponseElement
```python
class ResponseElementType(Enum):
    TEXT = "text"
    MARKDOWN = "markdown"
    CODE = "code"
    TABLE = "table"
    IMAGE = "image"
    CHART = "chart"
    REFERENCE = "reference"
    WARNING = "warning"
    INFO = "info"
    ERROR = "error"
    DIVIDER = "divider"
    LIST = "list"
    QUOTE = "quote"
```

### Conversión a Markdown
Cada elemento implementa `to_markdown()`:
- CODE → ```language\ncode\n```
- TABLE → Headers + Separators + Rows
- LIST → "- item" por cada item

## Consecuencias
- Positivas: Formateo consistente
- Positivas: Fácil extensión
- Neutrales: Mantenimiento de converters
