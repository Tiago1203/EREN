# ADR-0605: GE Healthcare Integration

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

GE Healthcare provides imaging and monitoring equipment. EREN needs to integrate with CARESTREAM PACS and monitoring systems.

## Decision

We will implement **GE Healthcare adapter**:

### Supported Systems

| System | Integration | Data |
|--------|-------------|------|
| CARESTREAM PACS | DICOM | Images |
| CARESCAPE | MQTT | Vital signs |
| Centricity | HL7 | Orders |

### Implementation

```python
class GEAdapter:
    async def fetch_imaging_worklist(self, date: date) -> list[WorkItem]:
        return await self.caris.get_worklist(date)
    
    async def subscribe_monitors(self, unit_id: str):
        await self.carescape.subscribe(f"/units/{unit_id}")
```

## Consequences

### Positive
- Imaging workflow
- Monitoring integration

### Negative
- Multiple protocols
- Version management
