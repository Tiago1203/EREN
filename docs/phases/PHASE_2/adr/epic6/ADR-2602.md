# ADR-2602: Streaming Support

## Estado
**Accepted** ✅

## Contexto
Se requiere soporte para streaming de respuestas.

## Decisión

### StreamingResponseComposer
```python
class StreamingResponseComposer(ResponseComposer):
    async def stream_text(text) -> AsyncIterator[StreamChunk]
    async def stream_markdown(markdown) -> AsyncIterator[StreamChunk]
    async def stream_response(response) -> AsyncIterator[StreamChunk]
```

### StreamChunk
```python
@dataclass
class StreamChunk:
    content: str
    is_final: bool
    chunk_type: str  # text, code, complete
```

### Estados de Response
```python
class ResponseStatus(Enum):
    PENDING = "pending"
    STREAMING = "streaming"
    COMPLETE = "complete"
    ERROR = "error"
    CANCELLED = "cancelled"
```

## Características
- Chunk size configurable
- Detección de bloques de código
- Actualización en tiempo real
- Estados de respuesta

## Consecuencias
- Positivas: Experiencia de usuario
- Positivas: Respuestas parciales
- Neutrales: Complejidad adicional
