# ADR-2501: Tool Execution

## Estado
**Accepted** ✅

## Contexto
La ejecución de herramientas necesita ser robusta y segura.

## Decisión

### ToolExecutor
```python
class ToolExecutor:
    max_concurrent: int = 10
    timeout_seconds: int = 30
    max_retries: int = 3
```

Características:
- Control de concurrencia con semaphores
- Timeout por ejecución
- Reintentos automáticos con backoff
- Tracking de ejecuciones

### ToolSandbox
```python
class ToolSandbox:
    allowed_paths: list[str]
    blocked_commands: list[str]
    max_memory_mb: int = 512
```

Características:
- Validación de paths
- Bloqueo de comandos peligrosos
- Ejecución Python aislada
- Verificación de permisos

## Consecuencias
- Positivas: Ejecución robusta
- Positivas: Seguridad
- Neutrales: Overhead de seguridad
