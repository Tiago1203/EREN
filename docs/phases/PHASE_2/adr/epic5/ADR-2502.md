# ADR-2502: Tool Routing

## Estado
**Accepted** ✅

## Contexto
El sistema debe seleccionar automáticamente la herramienta correcta.

## Decisión

### ToolRouter
Selección basada en:
1. **Rutas registradas**: intent -> tool_id
2. **Detección de categoría**: keywords en query
3. **Puntuación de relevancia**: nombre, descripción, tags

### Algoritmo
```python
def route(query, context):
    # 1. Buscar en rutas
    # 2. Detectar categoría por keywords
    # 3. Seleccionar mejor herramienta por score
```

### Categorías y Keywords
| Categoría | Keywords |
|-----------|----------|
| DATABASE | sql, query, select, datos |
| VECTOR_STORE | vector, embedding, similar |
| MESSAGE_QUEUE | cola, queue, mensaje |
| API | api, http, rest |
| CODE | código, python, script |

## Consecuencias
- Positivas: Routing automático
- Positivas: Extensible
- Neutrales: Puede fallar en casos ambiguos
