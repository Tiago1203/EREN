# ADR-2102: Event-Driven Conversation System

## Estado
**Accepted** ✅

## Contexto
Necesitamos notificar cambios en conversaciones a otros componentes del sistema.

## Decisión
Implementar sistema de eventos con ConversationEventDispatcher:

```python
class ConversationEventType(str, Enum):
    CREATED = "conversation.created"
    CLOSED = "conversation.closed"
    MESSAGE_ADDED = "conversation.message_added"
    PARTICIPANT_JOINED = "conversation.participant_joined"
    # ... más eventos
```

Patrón de suscripción:
```python
dispatcher.subscribe(ConversationEventType.CLOSED, handler)
dispatcher.dispatch(event)
```

## Consecuencias
- Positivas: Bajo acoplamiento
- Positivas: Fácil extensión
- Neutrales: Overhead de eventos
