# ADR-2301: Template System

## Estado
**Accepted** ✅

## Contexto
Los prompts necesitan ser reutilizables y configurables.

## Decisión
Implementar PromptTemplate con:

```python
class PromptTemplate:
    id: str
    name: str
    version: int
    system_template: str
    user_template: str
    variables: list[PromptVariable]
    examples: list[PromptExample]
    strategy: PromptStrategyType
```

Variables con sintaxis `{{variable}}` y `{{variable|default}}`.

## Consecuencias
- Positivas: Templates reutilizables
- Positivas: Validación de variables
- Positivas: Metadatos completos
