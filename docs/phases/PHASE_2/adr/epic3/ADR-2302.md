# ADR-2302: Prompt Strategies

## Estado
**Accepted** ✅

## Contexto
Diferentes tareas requieren diferentes estrategias de prompting.

## Decisión
Implementar estrategias como clases intercambiables:

| Estrategia | Descripción | Caso de uso |
|-----------|-------------|-------------|
| Direct | Pregunta directa | Simple queries |
| ChainOfThought | Razonamiento paso a paso | Problemas complejos |
| FewShot | Con ejemplos | Tareas específicas |
| TreeOfThought | Múltiples caminos | Creatividad |
| ReAct | Reason + Act | Con herramientas |
| SelfAsk | Preguntas intermedias | Clarificación |

## Consecuencias
- Positivas: Flexibilidad
- Positivas: Fácil extensión
- Neutrales: Overhead en selección
