# ADR-2101: AI Domain Gateway Pattern

**Status:** Accepted  
**Date:** 2026-07-20  
**Deciders:** Architecture Board  

---

## Context

AI Core necesita acceder a datos del dominio de negocio (Device, Incident, Knowledge, etc.) pero:
- AI Core no debe conocer implementaciones concretas del dominio
- El dominio no debe conocer al AI Core
- Necesitamos abstracciones explícitas para esta comunicación

---

## Decision

Implementar **Domain Gateways** como fachada oficial entre AI Core y Business Domain.

### Estructura

```
core/ai/domain/
├── contracts.py              # Interfaces abstractas
├── device_gateway.py         # Gateway para Device
├── incident_gateway.py       # Gateway para Incident
├── knowledge_gateway.py      # Gateway para Knowledge
├── recommendation_gateway.py # Gateway para Recommendation
├── hospital_gateway.py      # Gateway para Hospital
└── workorder_gateway.py     # Gateway para WorkOrder
```

### Contrato Base

```python
# contracts.py
from abc import ABC, abstractmethod
from typing import Any, Generic, TypeVar

T = TypeVar("T")

class IDomainGateway(ABC, Generic[T]):
    """Base interface for all domain gateways."""
    
    @abstractmethod
    async def get_by_id(self, id: str, tenant_id: str) -> T | None:
        """Get entity by ID."""
        raise NotImplementedError
    
    @abstractmethod
    async def search(
        self,
        query: str,
        tenant_id: str,
        filters: dict | None = None,
    ) -> list[T]:
        """Search entities."""
        raise NotImplementedError
```

from dataclasses import dataclass

@dataclass(frozen=True)
class DeviceDTO:
    """DTO for Device - exposed to AI."""
    id: str
    name: str
    device_type: str
    status: str
    is_critical: bool
    location: dict

class DeviceGateway(IDomainGateway[DeviceDTO]):
    """
    Gateway for Device domain.
    
    AI Core uses this gateway to access device data without
    knowing about repository implementations.
    """
    
    def __init__(
        self,
        device_repository: "DeviceRepository",  # Interface, not impl
        memory_manager: "MemoryManager",
    ):
        self._repository = device_repository
        self._memory = memory_manager
    
    async def get_by_id(
        self,
        id: str,
        tenant_id: str,
    ) -> DeviceDTO | None:
        """Get device by ID."""
        result = await self._repository.get_by_id(id, tenant_id)
        if result:
            return DeviceDTO.from_entity(result)
        return None

## Consequences

### Positive
- AI Core completamente desacoplado del dominio
- Easy to mock para tests
- Clear boundaries
- Gateway puede agregar datos de múltiples fuentes

### Negative
- Más capas de indirección
- Posible overhead de transformación DTO
