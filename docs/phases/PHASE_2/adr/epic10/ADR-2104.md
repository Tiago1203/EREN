# ADR-2104: Domain Access Rules

**Status:** Accepted  
**Date:** 2026-07-20  
**Deciders:** Architecture Board  

---

## Context

AI Core necesita acceder al dominio de negocio, pero debemos establecer reglas claras sobre qué puede y qué no puede acceder.

---

## Decision

Establecer reglas estrictas de acceso al dominio.

### Reglas de Acceso

#### AI PUEDE acceder (vía Gateways)

```python
# ✅ PERMITIDO
from core.ai.domain import DeviceGateway
from core.ai.context_builder.providers import DeviceProvider

# AI puede usar estos módulos
device_gateway = DeviceGateway()
incident_gateway = IncidentGateway()
knowledge_gateway = KnowledgeGateway()
recommendation_gateway = RecommendationGateway()
hospital_gateway = HospitalGateway()
```

#### AI NO PUEDE acceder (prohibido)

```python
# ❌ PROHIBIDO - Viola las reglas
from core.device.infrastructure import DeviceRepositoryImpl
from core.device.domain.repositories import DeviceRepository

# ❌ PROHIBIDO - Acceso directo a repositorio
class AIAccess:
    def __init__(self):
        self.repo = DeviceRepositoryImpl()  # NO!
```

### Módulo Restrictions

```
core/
├── device/
│   └── domain/
│       ├── repositories/        # ✅ AI puede usar interfaces
│       │   └── device_repository.py
│       └── infrastructure/      # ❌ AI NO puede usar
│           └── device_repository_impl.py
│
└── ai/
    └── domain/
        ├── device_gateway.py   # ✅ AI usa gateway
        └── contracts.py        # ✅ AI usa contratos
```

### Verificación en Tiempo de Ejecución

```python
# core/ai/domain/contracts.py
from functools import wraps

def domain_only(method):
    """
    Decorator para verificar que el caller es AI.
    
    En producción, esto verificaría el contexto de ejecución.
    """
    @wraps(method)
    async def wrapper(*args, **kwargs):
        # Verificar que el caller tiene permisos
        caller = get_current_context()
        if not caller.is_ai_core:
            raise PermissionError(
                f"Only AI Core can access domain gateways directly. "
                f"Use Application Services instead."
            )
        return await method(*args, **kwargs)
    return wrapper

class DeviceGateway:
    @domain_only
    async def get_by_id(self, id: str, tenant_id: str):
        ...
```

## Consequences

### Positive
- Clear boundaries enforced
- AI cannot bypass gateways
- Easy to audit access patterns
- Protection against accidental coupling

### Negative
- Enforcement adds complexity
- Runtime checks have overhead
