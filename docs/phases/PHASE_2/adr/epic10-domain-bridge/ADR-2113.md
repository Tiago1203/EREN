# ADR-2113: Entity Reference Serialization

**Status:** Proposed  
**Date:** 2026-07-20  
**Deciders:** Architecture Team  

---

## Context

Las memorias del AI Core necesitan guardar referencias a entidades del dominio de forma serializable para persistencia.

Requisitos:
1. Serializar referencias a strings
2. Recuperar entidades desde IDs
3. Mantener interoperabilidad con diferentes formatos

---

## Decision

Usar serialización JSON para DomainReference con formato estándar.

### Formato de Serialización

```python
# Serialización
{
    "type": "device",           # EntityType.value
    "id": "550e8400-e29b-...",  # UUID string
    "name": "Ventilator ICU-1", # Display name
    "metadata": {
        "device_type": "ventilator",
        "status": "active",
        ...
    }
}

# En contenido de memoria
"Device needs calibration. [Reference: {\"type\": \"device\", \"id\": \"...\", \"name\": \"Ventilator ICU-1\"}]"
```

### Implementación

```python
# core/ai/integration/serializers.py
from __future__ import annotations

import json
from dataclasses import dataclass, asdict
from datetime import datetime
from typing import Any, TYPE_CHECKING
from uuid import UUID

if TYPE_CHECKING:
    from core.PHASE_2.ai.integration.memory_bridge import DomainReference, EntityType


class EntityReferenceSerializer:
    """
    Serializador de DomainReference.
    
    Maneja la conversión entre DomainReference y diferentes formatos
    de serialización (JSON, string, etc.)
    """
    
    @staticmethod
    def to_json(reference: DomainReference) -> str:
        """
        Convierte referencia a JSON string.
        
        Args:
            reference: Referencia a serializar
            
        Returns:
            JSON string
        """
        data = {
            "type": reference.entity_type.value,
            "id": reference.entity_id,
            "name": reference.display_name,
            "metadata": reference.metadata,
        }
        return json.dumps(data)
    
    @staticmethod
    def from_json(data: str) -> DomainReference:
        """
        Crea referencia desde JSON string.
        
        Args:
            data: JSON string
            
        Returns:
            DomainReference
        """
        from core.PHASE_2.ai.integration.memory_bridge import DomainReference, EntityType
        
        parsed = json.loads(data)
        return DomainReference(
            entity_type=EntityType(parsed["type"]),
            entity_id=parsed["id"],
            display_name=parsed["name"],
            metadata=parsed.get("metadata", {}),
        )
    
    @staticmethod
    def to_compact_string(reference: DomainReference) -> str:
        """
        Convierte a string compacto para logging.
        
        Args:
            reference: Referencia a serializar
            
        Returns:
            String compacto "type:id:name"
        """
        return f"{reference.entity_type.value}:{reference.entity_id}:{reference.display_name}"
    
    @staticmethod
    def extract_from_content(content: str) -> list[DomainReference]:
        """
        Extrae todas las referencias de un contenido.
        
        Args:
            content: Contenido con referencias embebidas
            
        Returns:
            Lista de referencias encontradas
        """
        import re
        
        references = []
        # Patrones soportados
        patterns = [
            r'\[Reference:\s*(\{[^}]+\})\]',
            r'@ref\[([^\]]+)\]',
        ]
        
        for pattern in patterns:
            matches = re.findall(pattern, content)
            for match in matches:
                try:
                    ref = EntityReferenceSerializer.from_json(match)
                    references.append(ref)
                except (json.JSONDecodeError, KeyError):
                    continue
        
        return references
    
    @staticmethod
    def create_summary(references: list[DomainReference]) -> str:
        """
        Crea resumen de referencias para display.
        
        Args:
            references: Lista de referencias
            
        Returns:
            String formateado
        """
        if not references:
            return "No references"
        
        lines = []
        by_type: dict[str, list[DomainReference]] = {}
        
        for ref in references:
            if ref.entity_type.value not in by_type:
                by_type[ref.entity_type.value] = []
            by_type[ref.entity_type.value].append(ref)
        
        for entity_type, refs in by_type.items():
            lines.append(f"{entity_type.upper()} ({len(refs)}):")
            for ref in refs:
                lines.append(f"  - {ref.display_name} ({ref.entity_id[:8]}...)")
        
        return "\n".join(lines)


class DTOEncoder(json.JSONEncoder):
    """
    Encoder JSON custom para DTOs del AI Core.
    
    Usage:
        json.dumps(dto, cls=DTOEncoder)
    """
    
    def default(self, obj: Any) -> Any:
        from core.PHASE_2.ai.integration.memory_bridge import DomainReference
        from core.PHASE_2.ai.domain import DeviceDTO, IncidentDTO
        
        if isinstance(obj, DomainReference):
            return {
                "__type": "DomainReference",
                **asdict(obj),
            }
        
        if isinstance(obj, (DeviceDTO, IncidentDTO)):
            return {
                "__type": obj.__class__.__name__,
                **asdict(obj),
            }
        
        if isinstance(obj, UUID):
            return str(obj)
        
        if isinstance(obj, datetime):
            return obj.isoformat()
        
        return super().default(obj)
```

---

## Consequences

### Positive
- Formato estándar e interoperable
- Fácil debugging
- Recuperación de entidades desde IDs
- Compatible con persistencia

### Negative
- Overhead de parsing
- Dependencia de formato específico
