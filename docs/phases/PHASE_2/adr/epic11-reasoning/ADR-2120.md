# ADR-2120: Reasoning Engine

## Estado

**Aceptado** - 2026-07-20

## Contexto

Epic 11 implementa el Reasoning Engine — el motor que aplica estrategias de razonamiento sobre evidencia para alcanzar conclusiones con justificación auditable.

El Reasoning Engine es responsable de:
- Razonar sobre evidencia de múltiples fuentes
- Generar conclusiones justificables
- Calcular niveles de confianza
- Explicar el proceso de razonamiento

### Estrategias de Razonamiento

| Estrategia | Descripción | Caso de Uso |
|------------|-------------|-------------|
| Deductive | Premisas → Conclusión | Diagnóstico basado en reglas |
| Inductive | Observaciones → Patrón | Detección de anomalías |
| Abductive | Evidencia → Explicación | Razonamiento diagnóstico |
| Causal | Causas → Efectos | Análisis de fallos |

## Consecuencias

### Positivas
- Razonamiento explicable y auditable
- Múltiples estrategias disponibles
- Confianza calculada para cada conclusión

### Negativas
- Complejidad en la selección de estrategias
- Rendimiento puede verse afectado
