# ADR-2202: Context Compression & Prioritization

## Estado
**Accepted** ✅

## Contexto
El contexto puede crecer mucho. Necesitamos comprimir y priorizar.

## Decisión

### Compresión
Implementar estrategias:
- RemoveDuplicatesStrategy
- TrimWhitespaceStrategy
- RemoveRedundantInfoStrategy
- SummarizeStrategy (truncado inteligente)

### Priorización
Implementar ContextPrioritizer con:
- `relevance_weight`: Importancia del relevance_score
- `recency_weight`: Importancia de fecha
- `priority_weight`: Importancia de ContextPriority

Score = (relevance * 0.4) + (recency * 0.3) + (priority * 0.3)

## Consecuencias
- Positivas: Optimización de tokens
- Positivas: Mejor calidad de contexto
- Neutrales: Overhead en construcción
