# ADR-2201: Context Window Management

## Estado
**Accepted** ✅

## Contexto
El LLM tiene límites de tokens. Necesitamos gestionar qué contexto incluir.

## Decisión
Implementar ContextWindow con:
- `max_tokens`: Límite total
- `min_critical_tokens`: Mínimo para info crítica
- Items priorizados por `ContextPriority`

Prioridades:
```python
CRITICAL = 1    # Siempre incluir
HIGH = 2        # Importante
MEDIUM = 3      # Estándar
LOW = 4         # Omitible
DISCARDABLE = 5 # Puede descartarse
```

## Consecuencias
- Positivas: Control preciso de tokens
- Positivas: Garantía de contexto mínimo
- Neutrales: Requiere calibración de prioridades
