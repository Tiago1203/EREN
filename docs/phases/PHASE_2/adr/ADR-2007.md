# ADR-2007: DI Strategy

## Estado
**Accepted** ✅

## Contexto
Necesitamos un sistema de inyección de dependencias para el AI Core.

## Decisión
Implementar `ContainerImpl` con:

- Registro de dependencias (singleton o transient)
- Resolución de dependencias
- Scopes para ciclo de vida
- Decoradores `@injectable`

## Consecuencias
- Positivas: Bajo acoplamiento
- Positivas: Fácil testing
- Neutrales: Curva de aprendizaje
