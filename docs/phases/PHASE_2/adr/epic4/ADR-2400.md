# ADR-2400: Memory Architecture

## Estado
**Accepted** ✅

## Contexto
Necesitamos un sistema de memoria que gestione diferentes tipos de recuerdos.

## Decisión
Implementar MemoryManager con múltiples tipos:

```python
class MemoryType(str, Enum):
    WORKING = "working"     # Sesión activa
    SHORT = "short"         # Corto plazo
    LONG = "long"           # Largo plazo
    EPISODIC = "episodic"  # Experiencias
    SEMANTIC = "semantic"   # Conocimiento
```

Estructura:
```
core/ai/memory/
├── models.py         # Modelos de memoria
├── repository.py    # Repositorio abstracto
└── manager.py      # MemoryManager
```

## Consecuencias
- Positivas: Organización clara de memorias
- Positivas: Límites por tipo
- Neutrales: Complejidad adicional
