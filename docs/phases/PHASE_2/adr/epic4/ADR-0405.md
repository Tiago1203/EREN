# ADR-0405: Reasoning Engine Strategy

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

EREN necesita un motor de razonamiento que pueda:
1. **Razonar paso a paso** — Chain-of-Thought
2. **Explorar múltiples paths** — Tree-of-Thought
3. **Usar herramientas** — ReAct (Reason + Act)
4. **Planificar** — Decompose y ejecutar tareas complejas

Epic 0 Cognitive Model define las capabilities de razonamiento. Este ADR define la estrategia de implementación.

---

class ReasoningMode(Enum):
    DIRECT = "direct"
    CHAIN_OF_THOUGHT = "cot"
    TREE_OF_THOUGHT = "tot"
    REACT = "react"
    PLAN = "plan"

**Chain-of-Thought (COT):**

```python
async def reason_cot(
    self,
    context: ReasoningContext,
    evidence: list[Evidence],
) -> ReasoningResult:
    """
    1. Generate reasoning steps
    2. Each step builds on previous
    3. Final answer synthesized
    """
    prompt = self.prompt_builder.build_cot_prompt(context, evidence)
    
    response = await self.llm.complete(
        prompt=prompt,
        stop_sequences=["Final Answer:"],
    )
    
    steps = self.parse_reasoning_steps(response.content)
    
    return ReasoningResult(
        mode=ReasoningMode.CHAIN_OF_THOUGHT,
        steps=steps,
        final_answer=steps[-1].conclusion,
        trace=response.reasoning_content,  # If supported
    )
```

**Tree-of-Thought (TOT):**

```python
async def reason_tot(
    self,
    context: ReasoningContext,
    evidence: list[Evidence],
    branches: int = 3,
) -> ReasoningResult:
    """
    1. Generate N branches of reasoning
    2. Evaluate each branch
    3. Select best or combine
    """
    branches = []
    
    for _ in range(branches):
        branch = await self.explore_branch(context, evidence)
        score = await self.evaluate_branch(branch)
        branches.append((branch, score))
    
    # Select or combine
    best = max(branches, key=lambda x: x[1])
    
    return ReasoningResult(
        mode=ReasoningMode.TREE_OF_THOUGHT,
        branches=branches,
        selected_branch=best[0],
        selection_rationale=f"Score: {best[1]}",
    )
```

**ReAct (Reason + Act):**

```python
async def reason_react(
    self,
    context: ReasoningContext,
    evidence: list[Evidence],
    max_iterations: int = 5,
) -> ReasoningResult:
    """
    Loop:
    1. Reason about next action
    2. Execute action (via Tool Orchestrator)
    3. Observe result
    4. Repeat until done
    """
    observations = []
    history = []
    
    for i in range(max_iterations):
        # Reason
        thought = await self.reason_step(
            context=context,
            evidence=evidence,
            observations=observations,
            history=history,
        )
        
        if thought.is_final:
            return ReasoningResult(
                mode=ReasoningMode.REACT,
                steps=history,
                final_answer=thought.conclusion,
            )
        
        # Act
        if thought.action:
            result = await self.tool_orchestrator.execute(thought.action)
            observations.append(result)
            history.append(ThoughtStep(thought=thought, action=thought.action, observation=result))
        
        # If no action but not final, need more context
        if not thought.action and not thought.is_final:
            observations.append(Thought(text="Need more information"))
            history.append(ThoughtStep(thought=thought, observation="No action taken"))
```

## Mode Selection

```python
async def reason(
    self,
    context: ReasoningContext,
    evidence: list[Evidence],
) -> ReasoningResult:
    """
    Automatically select reasoning mode based on:
    1. Query complexity
    2. Available tools
    3. Time constraints
    4. User preferences
    """
    complexity = await self.estimate_complexity(context.query)
    
    if complexity == QueryComplexity.SIMPLE:
        return await self.reason_direct(context, evidence)
    
    elif complexity == QueryComplexity.MODERATE:
        return await self.reason_cot(context, evidence)
    
    elif complexity == QueryComplexity.COMPLEX:
        if context.available_tools:
            return await self.reason_react(context, evidence)
        return await self.reason_tot(context, evidence)
    
    elif complexity == QueryComplexity.VERY_COMPLEX:
        return await self.reason_plan(context, evidence)
```

---

## Reasons

**1. Flexibility:**
- Different queries need different reasoning strategies
- Simple queries: fast direct response
- Complex queries: thorough exploration

**2. EREN principles:**
- Transparency: COT shows reasoning
- Evidence: RAG provides grounding
- Safety: Multiple modes allow verification

**3. Performance:**
- Fast mode for simple queries
- Slow mode only when needed
- Budget-aware execution

---

## Consequences

**Positive:**
- Adaptive reasoning quality
- Transparent reasoning traces
- Tool use capability
- Handles complexity variation

**Negative:**
- More complex than single-mode
- Latency varies by mode
- Harder to test all paths
