# ADR-2003: Exception Hierarchy

## Estado
**Accepted** ✅

## Contexto
Necesitamos una jerarquía de excepciones específica para el AI Core.

## Decisión
Implementar jerarquía de excepciones:

```
AIError (base)
├── AIConfigurationError
├── AIProviderError
│   ├── AIProviderNotFoundError
│   ├── AIRateLimitError
│   ├── AIAuthenticationError
│   └── AIQuotaExceededError
├── AIModelError
│   └── AIModelNotFoundError
├── AIContextError
├── AIValidationError
├── AIInjectionError
├── AIInitializationError
├── AIProcessingError
└── AITimeoutError
```

## Consecuencias
- Positivas: Error handling específico
- Positivas: Códigos de error únicos
- Neutrales: Puede ser verboso
