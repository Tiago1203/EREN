# ADR-2008: Context Object Design

## Estado
**Accepted** ✅

## Contexto
Necesitamos objetos de contexto para requests, conversaciones y sesiones.

## Decisión
Implementar:

- `RequestContext`: Contexto de request individual
- `ConversationContext`: Contexto de conversación
- `SessionContext`: Contexto de sesión
- `AIContextManager`: Gestor de contextos con TTL

## Consecuencias
- Positivas: Aislamiento de requests
- Positivas: Limpieza automática
- Neutrales: Overhead de memoria
