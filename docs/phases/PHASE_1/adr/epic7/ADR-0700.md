# ADR-0700: User Experience Architecture

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

EPIC 7 implements the User Experience layer for EREN — the interface that brings all backend services together.

## Dependencies

- **EPIC 0** — Philosophy, Architecture
- **EPIC 1** — Infrastructure Platform
- **EPIC 2** — Domain Models (Device, Incident)
- **EPIC 3** — Hospital Contexts
- **EPIC 4** — AI Core (Cognitive Runtime)
- **EPIC 5** — Clinical Intelligence (CDSS)
- **EPIC 6** — Integrations

## Prerequisite for

- **EPIC 8** — Production Readiness (necesita UX estable y lista)
- **EPIC 9** — Machine Learning (feedback loop de usuarios)

## Decision

We will implement a **Multi-Platform User Experience**:

```
┌─────────────────────────────────────────────────────────────┐
│                    User Experience Layer                       │
│                                                               │
│  ┌─────────────┐  ┌──────────────┐  ┌─────────────────┐   │
│  │   Web App   │  │  Mobile App  │  │   Accessibility │   │
│  │  (Next.js)  │  │  (React Nat)│  │     (WCAG)      │   │
│  └─────────────┘  └──────────────┘  └─────────────────┘   │
└───────────────────────────┬─────────────────────────────────┘
                            │
         ┌──────────────────┼──────────────────┐
         ▼                  ▼                  ▼
┌─────────────────┐  ┌──────────┐     ┌──────────────┐
│     EPIC 6       │  │  EPIC 5  │     │    EPIC 4    │
│   Integrations   │  │  Clinical │     │   AI Core     │
│                  │  │Intelligence│     │              │
└─────────────────┘  └──────────┘     └──────────────┘
```

## Components

1. **Web Application** — Next.js dashboard
2. **Mobile Application** — React Native (iOS/Android)
3. **Chat Interface** — AI-powered conversation
4. **Dashboards** — Real-time metrics
5. **Accessibility** — WCAG 2.1 AA compliance
