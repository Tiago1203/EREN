# ADR-0701: Web Application Architecture

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

The web application is the primary interface for EREN. We need a modern, scalable, and accessible UI.

## Decision

We will use **Next.js 14** with:

### Technology Stack

| Component | Technology |
|-----------|------------|
| Framework | Next.js 14 (App Router) |
| Language | TypeScript |
| Styling | Tailwind CSS |
| State | Zustand + React Query |
| Forms | React Hook Form + Zod |
| UI Components | shadcn/ui |
| Charts | Recharts |

### Key Features

1. **Server-Side Rendering** — SEO and performance
2. **Static Generation** — Fast page loads
3. **API Routes** — Backend-for-Frontend pattern
4. **Real-time** — WebSocket integration
