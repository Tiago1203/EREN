# ADR-0708: Accessibility Standards

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Accessibility ensures EREN is usable by all engineers, including those with disabilities.

## Decision

We will comply with **WCAG 2.1 Level AA**:

### Guidelines

| Guideline | Implementation |
|-----------|---------------|
| Perceivable | Alt text, captions, contrast 4.5:1 |
| Operable | Keyboard navigation, focus indicators |
| Understandable | Clear language, error messages |
| Robust | Semantic HTML, ARIA labels |

### Testing

1. **Automated** — axe-core, Lighthouse
2. **Manual** — Screen reader testing
3. **User Testing** — With disabled users

### Features

- Screen reader support
- Keyboard shortcuts
- High contrast mode
- Font scaling
- Reduced motion
