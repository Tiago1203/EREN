# ADR-0707: Mobile Application Architecture

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Mobile application provides on-the-go access to EREN for field engineers.

## Decision

We will implement **React Native** application:

### Technology Stack

| Component | Technology |
|-----------|------------|
| Framework | React Native 0.73 |
| Language | TypeScript |
| State | Zustand |
| Navigation | React Navigation 6 |
| Offline | WatermelonDB |
| Push | Firebase Cloud Messaging |

### Supported Platforms

1. **iOS 14+** — App Store
2. **Android 8+** — Play Store + APK

### Key Features

- Offline mode with sync
- Push notifications
- Camera/barcode scanning
- Voice input
