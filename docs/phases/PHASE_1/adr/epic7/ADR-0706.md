# ADR-0706: Recommendations UI

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Recommendations UI surfaces AI-generated suggestions for device management.

## Decision

We will implement a recommendations hub:

### Features

1. **Priority Queue** — Sorted by confidence
2. **Detail View** — Evidence and reasoning
3. **Action Buttons** — Accept, defer, dismiss
4. **Feedback Loop** — User corrections improve AI

### Recommendation Types

| Type | Trigger | Action |
|------|---------|--------|
| Maintenance | Scheduled | Preventive action |
| Calibration | Drift detected | Recalibrate |
| Replacement | Age/Risk | Order parts |
| Alert | Anomaly | Investigate |
