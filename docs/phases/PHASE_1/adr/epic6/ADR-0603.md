# ADR-0603: DICOM Integration

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

DICOM is the standard for medical imaging. EREN needs to manage imaging device metadata and connect to PACS systems.

## Decision

We will implement **DICOM client capabilities**:

### Supported Operations

| Service | Use Case |
|---------|----------|
| C-FIND | Search images |
| C-MOVE | Retrieve images |
| C-STORE | Store reports |
| DICOMDIR | Media management |

### Implementation

```python
class DICOMClient:
    async def search_images(
        self,
        modality: str,
        date_range: tuple[date, date],
    ) -> list[DICOMImage]:
        result = await self.c_find({
            "Modality": modality,
            "StudyDate": f"{start}-{end}",
        })
        return result
    
    async def get_device_info(self, ae_title: str) -> DeviceInfo:
        return await self.query_node(ae_title)
```

## Consequences

### Positive
- Imaging device integration
- PACS connectivity

### Negative
- DICOM complexity
- Large data handling
