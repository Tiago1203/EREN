# ADR-0611: Azure AD Integration

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Azure AD provides identity and access management. EREN needs SSO and directory synchronization.

## Decision

We will implement **Azure AD (Microsoft Entra) adapter**:

### Supported Operations

| Operation | Use Case |
|-----------|----------|
| OAuth 2.0 | SSO authentication |
| Graph API | User directory sync |
| Groups | Role assignment |
| Licenses | Feature access |

### Implementation

```python
class AzureADAdapter:
    async def get_user(self, user_id: str) -> User:
        graph = GraphClient(self.token)
        return await graph.users[user_id].get()
        
    async def get_user_groups(self, user_id: str) -> list[str]:
        graph = GraphClient(self.token)
        groups = await graph.users[user_id].member_of.get()
        return [g.display_name for g in groups]
```

## Consequences

### Positive
- Enterprise SSO
- User management

### Negative
- Cloud dependency
- Permission scopes
