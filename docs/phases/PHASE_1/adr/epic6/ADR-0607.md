# ADR-0607: Mindray Integration

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Mindray provides patient monitors and imaging equipment. EREN needs to integrate for telemetry and maintenance.

## Decision

We will implement **Mindray eGateway adapter**:

### Supported Equipment

| Equipment | Data | Frequency |
|-----------|------|-----------|
| BeneVision N | Vitals | Real-time |
| PM-8000 | Vitals | Real-time |
| DC-8 | Ultrasound | Event |
| TE7 | Ultrasound | Event |

### Implementation

```python
class MindrayAdapter:
    async def connect_egateway(self, host: str):
        self.egateway = MindrayEGateway(host)
        await self.egateway.authenticate()
        
    async def subscribe_patients(self):
        await self.egateway.subscribe_all(self.on_patient_data)
        
    async def on_patient_data(self, data: PatientData):
        await self.process_vitals(data)
```

## Consequences

### Positive
- Cost-effective integration
- Wide device support

### Negative
- Limited documentation
- API restrictions
