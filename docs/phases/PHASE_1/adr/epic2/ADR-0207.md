# ADR-0207: Knowledge Article Value Objects

**Status:** Accepted

**Date:** 2026-07-16

**Deciders:** Architecture Board

---

## Context

Knowledge Article is the aggregate root of the Knowledge bounded context. We need to define its value objects and invariants.

---

## Decision

### Aggregate Root: KnowledgeArticle

```python
@dataclass(eq=False)
class KnowledgeArticle(AggregateRoot):
    """Knowledge Article aggregate root."""

    tenant_id: TenantId
    title: str
    content: KnowledgeContent
    category: ArticleCategory
    tags: list[str]
    device_id: DeviceId | None  # Optional link to device
    author: EngineerId
    version: int

    # State
    status: ArticleStatus = field(default_factory=ArticleStatus.draft)
    published_at: datetime | None = None
    archived_at: datetime | None = None
```

### Value Objects

```python
@dataclass(frozen=True)
class ArticleCategory(ValueObject):
    """Category of knowledge article."""
    value: str  # device_manual | maintenance_procedure | safety_alert |
                # regulatory | clinical_protocol | troubleshooting_guide

    VALID = {
        "device_manual",
        "maintenance_procedure",
        "safety_alert",
        "regulatory",
        "clinical_protocol",
        "troubleshooting_guide",
        "training_material",
        "incident_report",
    }

@dataclass(frozen=True)
class ArticleStatus(ValueObject):
    """Status of knowledge article."""
    value: str  # draft | published | archived | retired

    VALID = {"draft", "published", "archived", "retired"}

@dataclass(frozen=True)
class KnowledgeContent(ValueObject):
    """Content of a knowledge article with metadata."""
    body: str
    summary: str | None
    references: list[str]  # URLs or document IDs
    citations: list[str]     # Evidence IDs

    def __post_init__(self) -> None:
        if not self.body or len(self.body.strip()) < 10:
            raise ValueError("Article body must be at least 10 characters")
```

### Invariants

1. Body must be at least 10 characters
2. Category must be one of the VALID values
3. Status must be one of the VALID values
4. Cannot publish without a body
5. Archived articles cannot be republished
6. Tags must be non-empty list (can be empty list, not None)
7. Version starts at 1 and increments on update

### Lifecycle

```
DRAFT → PUBLISHED → ARCHIVED
           ↘ RETIRED
```

**State transitions:**
- draft → published: requires non-empty body
- published → archived: manual or auto (no citations)
- archived → retired: only by admin
- retired: terminal state

---

## Consequences

**Positive:**
- Clear model for knowledge articles
- Validation enforced at creation time
- Supports evidence citation workflow

**Negative:**
- May need to add more categories as domain evolves
- No versioning yet (consider v2 articles in future ADR)
