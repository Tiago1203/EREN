# ADR-0806: ISO 27001 Compliance

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

ISO 27001 certification demonstrates commitment to information security management.

## Decision

We will implement **ISO 27001 Controls**:

### Control Domains

| Domain | Controls | Implementation |
|--------|----------|----------------|
| A.5 | Information Security Policies | Documentation |
| A.6 | Organization | Roles + Responsibilities |
| A.7 | Human Resource | Security training |
| A.8 | Asset Management | Classification + Handling |
| A.9 | Access Control | RBAC + MFA |
| A.10 | Cryptography | Encryption standards |
| A.12 | Operations Security | Patch management |
| A.13 | Communications | Network security |
| A.14 | System Acquisition | Security requirements |
| A.16 | Incident Management | Response procedures |

### Key Controls

```yaml
iso27001_controls:
  access_control:
    - policy: password_policy
    - policy: mfa_required
    - policy: session_timeout_15m
  cryptography:
    - algorithm: AES-256
    - key_rotation: 90_days
  incident_response:
    - notification: 24h
    - escalation: predefined
```
