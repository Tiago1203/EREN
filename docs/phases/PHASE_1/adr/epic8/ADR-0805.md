# ADR-0805: GDPR Compliance

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

GDPR compliance is required for handling EU citizen data.

## Decision

We will implement **GDPR Data Subject Rights**:

### Rights Implemented

| Right | Implementation |
|-------|---------------|
| Right to Access | User data export |
| Right to Rectification | Profile editing |
| Right to Erasure | Data deletion pipeline |
| Right to Portability | JSON export |
| Right to Object | Opt-out mechanisms |
| Data Minimization | Collect only necessary |

### Implementation

```python
class GDPRService:
    async def export_user_data(self, user_id: str) -> dict:
        """Right to Portability"""
        return await self.data_export(user_id, format="json")
    
    async def delete_user_data(self, user_id: str) -> bool:
        """Right to Erasure"""
        await self.anonymize_pii(user_id)
        await self.delete_non_essential(user_id)
        return True
```

### Data Retention

| Data Type | Retention | Legal Basis |
|-----------|-----------|-------------|
| Logs | 7 years | Legal obligation |
| Audit | 7 years | Compliance |
| User Data | Until deletion | Consent |
| Analytics | 2 years | Legitimate interest |
