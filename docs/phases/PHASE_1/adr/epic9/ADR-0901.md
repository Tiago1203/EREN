# ADR-0901: Feedback Learning System

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

El sistema necesita aprender del feedback de usuarios para mejorar continuamente las recomendaciones y respuestas del AI.

---

## Decision

Implementar un sistema de Feedback Learning con:

1. **Explicit Feedback** — Ratings, thumbs up/down, corrections
2. **Implicit Feedback** — Time on response, follow-up questions, task completion
3. **Structured Feedback** — Categorized corrections by type

```
User Feedback → Collector → Processor → Learning Engine → Knowledge Update
     │                                                    │
     └────────────── Feedback Store ←─────────────────────┘
```

---

## Consequences

**Pros:**
- Mejora continua del sistema
- Datos de entrenamiento de alta calidad
- Feedback categorizado

**Cons:**
- RequiereVolume suficiente de feedback
- Posible sesgo en usuarios activos
