# ADR-1003: Helm Chart Design

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Helm charts para deployment en Kubernetes.

---

## Decision

Helm chart structure:

```
eren/
├── Chart.yaml
├── values.yaml
├── values.schema.json
├── README.md
├── templates/
│   ├── NOTES.txt
│   ├── _helpers.tpl
│   ├── deployment.yaml
│   ├── service.yaml
│   ├── configmap.yaml
│   ├── secrets.yaml
│   ├── hpa.yaml
│   └── ingress.yaml
└── .helmignore
```

---

## Consequences

**Pros:**
- Deploy reproducible
- Versioning de config
- Rollback fácil

**Cons:**
- Curva de aprendizaje
- Template complexity
