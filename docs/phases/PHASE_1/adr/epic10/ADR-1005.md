# ADR-1005: Versioning Strategy

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Estrategia de versionado semántico.

---

## Decision

Semantic versioning (SemVer):

| Version | Description |
|---------|-------------|
| Major | Breaking changes |
| Minor | New features |
| Patch | Bug fixes |

Format: `MAJOR.MINOR.PATCH` (e.g., 1.0.0)

---

## Consequences

**Pros:**
- Clear versioning
- Predictable upgrades
- Changelog generation

**Cons:**
- Strict rules
- Release management
