# ADR-0076: Model Versioning Strategy

**Status:** Proposed

**Date:** 2026-07-16

**Deciders:** AI Team

---

## Context

CDS models must be versioned for reproducibility, rollback capability, and regulatory compliance.

---

## Decision

**PENDING — Will be completed during EPIC 9.**

### Versioning Strategy

| Artifact | Versioning |
|---------|-----------|
| LLM prompts | Git versioned |
| Embedding model | Model registry |
| Fine-tuned models | Model registry |
| RAG knowledge base | Dated snapshots |
| CDS confidence thresholds | Config versioned |
