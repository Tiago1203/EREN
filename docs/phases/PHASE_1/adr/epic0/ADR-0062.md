# ADR-0062: AI Explainability Requirements

**Status:** Accepted

**Date:** 2026-07-16

**Deciders:** Architecture Board

---

## Context

Clinical users must understand why EREN made a recommendation. "Black box" AI is not acceptable in clinical settings.

---

## Decision

**Every CDS recommendation must include a human-readable explanation.**

The explanation must include:
1. **What** — The recommendation
2. **Why** — The reasoning chain from evidence to conclusion
3. **How certain** — The confidence score and contributing factors
4. **What might change it** — What additional information would change the recommendation
5. **Sources** — Evidence citations

---

## Explanation Quality

| Element | Required |
|---------|---------|
| Reasoning chain | Yes — step-by-step logic |
| Confidence score | Yes — 0.0-1.0 |
| Evidence citations | Yes — at least one |
| Uncertainty statement | Yes — if confidence < 0.5 |
| Alternative hypotheses | Yes — if differential diagnosis |
