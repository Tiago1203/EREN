# ADR-0032: Circuit Breaker Pattern

**Status:** Deprecated

**Date:** 2026-05-15

**Deciders:** Infrastructure Team

---

## Context

External service failures should not cascade into system-wide failures.

---

## Decision

**DEPRECATED.** This is a standard software pattern, not a specific architectural decision.

The Circuit Breaker pattern is documented in `EREN_FAILURE_MODEL.md` (Epic 0.5) and `ADR-0085_OBSERVABILITY.md` (Epic 0-Infra) as a component configuration, not as a standalone ADR.

**This ADR is deprecated. No action needed — the pattern lives in implementation code.**

---

## Why Deprecated

1. **Not EREN-specific:** Circuit Breaker is a well-known pattern (Martin Fowler, 2014)
2. **Too generic:** Any microservice architecture uses it — it's implementation, not architecture
3. **Already documented:** `EREN_FAILURE_MODEL.md` has the circuit breaker configuration
4. **Confusing:** Creating an ADR for a standard pattern pollutes the ADR index

---
