# ADR-0075: ML Feedback Loop Architecture

**Status:** Proposed

**Date:** 2026-07-16

**Deciders:** AI Team

---

## Context

EREN's AI models (CDS recommendations, failure predictions) must improve over time based on user feedback and outcomes.

---

## Decision

**PENDING — Will be completed during EPIC 9.**

### Feedback Loop Architecture

```
User Feedback (accept/reject/modify)
    │
    ▼
Feedback Storage (PostgreSQL)
    │
    ▼
Analytics (weekly aggregation)
    │
    ▼
Model Evaluation (quarterly)
    │
    ▼
Fine-tuning or Prompt Optimization
    │
    ▼
New Model Version Deployed
```

---

## Feedback Types

| Feedback | Use Case |
|---------|---------|
| Recommendation accepted | Positive signal for CDS |
| Recommendation rejected | Negative signal |
| Recommendation modified | Strong negative signal |
| Feedback reason | Fine-tuning data |
