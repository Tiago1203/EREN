# ADR-0066: Mobile App Architecture

**Status:** Proposed

**Date:** 2026-07-16

**Deciders:** Mobile Team

---

## Context

EREN needs mobile applications for iOS and Android for on-the-go access to alarms, patient data, and maintenance tasks.

---

## Decision

**PENDING — Will be decided during EPIC 7.**

### Current Considerations

| Option | Pros | Cons |
|--------|------|------|
| React Native | Shared code, JS ecosystem | Native performance |
| Flutter | Performance, UI quality | Dart learning curve |
| Native (Swift/Kotlin) | Best performance | Two codebases |
