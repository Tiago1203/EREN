# ADR-0070: GDPR Compliance Strategy

**Status:** Proposed

**Date:** 2026-07-16

**Deciders:** Security Team

---

## Context

EREN may serve hospitals in EU countries subject to GDPR. GDPR requires specific controls for personal data of EU residents.

---

## Decision

**PENDING — Will be completed during EPIC 8.**

### GDPR Requirements to Address

| Requirement | Implementation |
|-----------|---------------|
| Right to Erasure | Delete patient data on request (cascade to all stores) |
| Data Portability | Export patient data in machine-readable format |
| Consent Management | Track consent for data processing |
| Data Minimization | Only collect data necessary for care |
| Breach Notification | 72-hour notification to DPA |
| Data Residency | EU patient data stored in EU data centers |
