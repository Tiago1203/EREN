    @classmethod

### Principios

1. **Inmutabilidad**: @frozen dataclass
2. **Solo datos necesarios**: No exponer todo
3. **Factory method**: from_entity() es la Гєnica forma de crear
4. **Sin lГіgica**: Solo datos, ningГєn comportamiento
5. **Tipos simples**: str, int, float, dict, list, datetime
