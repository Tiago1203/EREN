# ADR-11005: Dependency Inversion

**Status:** Proposed  
**Date:** 2026-07-20  
**Deciders:** Architecture Board  

---

## Context

AI Core (FASE 2) necesita acceder al dominio (FASE 1), pero:
- AI Core no debe depender de implementaciones concretas
- El dominio no debe conocer a AI Core
- Necesitamos abstracciones compartidas

---

## Decision

Aplicar **Dependency Inversion Principle** (DIP) de SOLID.

### Principio

```
┌────────────────────────────────────────────────────────────┐
│  ❌ ANTES (Acoplamiento directo)                           │
│                                                             │
│  AI Core ──────▶ DeviceRepositoryImpl                     │
│                    (violación de DIP)                       │
└────────────────────────────────────────────────────────────┘

┌────────────────────────────────────────────────────────────┐
│  ✅ DESPUÉS (Abstracción)                                  │
│                                                             │
│  AI Core ──────▶ IDeviceQueryService ──────▶ DeviceQuerySvc│
│                    (interfaz)         (implementación)     │
└────────────────────────────────────────────────────────────┘
```

### Interfaces en Shared

```
core/shared/services/
├── query/
│   └── interfaces.py      # IQueryService<T>
├── command/
│   └── interfaces.py      # ICommandService<T, C>
└── repositories/
    └── interfaces.py       # IRepository<T, Id>
```

### Ejemplo

```python
# core/shared/services/query/interfaces.py
from abc import ABC, abstractmethod
from typing import TypeVar, Generic

T = TypeVar("T")

class IQueryService(ABC, Generic[T]):
    """Abstracción para servicios de query."""
    
    @abstractmethod
    async def get_by_id(self, id: str) -> T | None:
        ...

# core/application/services/query/device_query_service.py
from core.shared.services.query.interfaces import IQueryService

class DeviceQueryService(IQueryService[DeviceDTO]):
    """Implementación concreta."""
    
    def __init__(self, repository: DeviceRepository):
        self._repository = repository
    
    async def get_by_id(self, id: str) -> DeviceDTO | None:
        result = await self._repository.get_by_id(id)
        return DeviceDTO.from_entity(result) if result else None

# core/ai/tools/device_tool.py (AI Core)
from core.application.services.query import IDeviceQueryService

class DeviceTool:
    """Tool para AI Core."""
    
    def __init__(self, query_service: IDeviceQueryService):
        # AI Core solo conoce la interfaz, no la implementación
        self._query_service = query_service
```

## Consequences

### Positive
- AI Core desacoplado de implementaciones
- Fácil de mockear para tests
- Swap de implementaciones sin cambios en AI

### Negative
- Más interfaces a definir
- Indirección adicional
