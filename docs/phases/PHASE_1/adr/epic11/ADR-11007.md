# ADR-11007: Shared Interfaces Location

**Status:** Proposed  
**Date:** 2026-07-20  
**Deciders:** Architecture Board  

---

## Context

Necesitamos definir dónde se crean las interfaces compartidas entre dominio y AI Core.

---

## Decision

Crear un módulo `core/shared/services/` para interfaces compartidas.

### Estructura

```
core/shared/
├── entities/          # BaseEntity, AggregateRoot
├── events/           # DomainEvent
├── primitives/       # EntityId types
├── value_objects/    # Value Objects compartidos
└── services/         # NUEVO: Interfaces de servicios
    ├── __init__.py
    ├── query/
    │   └── interfaces.py      # IQueryService<T>
    ├── command/
    │   └── interfaces.py      # ICommandService<T, C>
    └── repositories/
        └── interfaces.py      # IRepository<T, Id>
```

### Interfaces a Definir

```python
# core/shared/services/query/interfaces.py
from abc import ABC, abstractmethod
from typing import Generic, TypeVar, Optional

Result = TypeVar("Result")
T = TypeVar("T")

class IQueryService(ABC, Generic[T]):
    """Base interface for all Query Services."""
    
    @abstractmethod
    async def get_by_id(self, id: str) -> Optional[T]:
        """Get entity by ID."""
        raise NotImplementedError
    
    @abstractmethod
    async def list_all(
        self,
        limit: int = 100,
        offset: int = 0,
    ) -> list[T]:
        """List all entities."""
        raise NotImplementedError
```

```python
# core/shared/services/command/interfaces.py
from abc import ABC, abstractmethod
from typing import Generic, TypeVar

T = TypeVar("T")
C = TypeVar("C")

class ICommandService(ABC, Generic[T, C]):
    """Base interface for all Command Services."""
    
    @abstractmethod
    async def execute(self, command: C) -> T:
        """Execute command and return result."""
        raise NotImplementedError
```

---

## Consequences

### Positive
- Interfaces en ubicación compartida
- Fácil de importar desde cualquier módulo
- Clear ownership (shared kernel)

### Negative
- Más módulos en shared
- Posible scope creep en shared
