# EREN Epic 2 — Core Business Domain

*Version 1.1 - 2026-07-20*

Epic 2 implements the core business domain with Device, Incident, Recommendation, and Knowledge bounded contexts.

---

## Components

### Device Context
- Device aggregate with full lifecycle
- DeviceType, Manufacturer, Location value objects

### Incident Context
- Incident aggregate with investigation workflow
- Severity levels, priority calculation

### Recommendation Context
- Recommendation entity with evidence tracking
- Confidence scoring

### Knowledge Context
- Knowledge article management
- Category taxonomy
