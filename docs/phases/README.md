## 🚀 Empezar Nueva Fase

1. Leer el documento de la fase en `docs/phases/PHASE_X_*.md`
2. Revisar ADRs en `docs/adr/epicX/`
3. Revisar documentación en `docs/epicX/`
4. Implementar código en `apps/api/app/` o `core/`
5. Crear PR
