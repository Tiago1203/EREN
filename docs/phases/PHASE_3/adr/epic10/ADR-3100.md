# ADR-3100: Learning Engine Architecture

**Título:** Arquitectura del Motor de Aprendizaje
**Estado:** ✅ COMPLETO
**Fecha:** 2026-07-21
**Decisor:** Architecture Team

## Contexto

El Learning Engine cierra el ciclo de inteligencia de EREN. No razona ni responde usuarios, sino que aprende de los resultados y actualiza el conocimiento.

### Learning Package

```python
@dataclass
class LearningPackage:
    """Package of learnings to distribute."""
    new_knowledge: list[dict]
    updated_confidence: list[dict]
    new_patterns: list[dict]
    suggested_rules: list[dict]
    recommendation_scores: list[dict]
    similar_cases: list[dict]
    experience_records: list[dict]
    learning_metrics: dict


class LearningEngine:
    """Main learning engine."""
    
    def __init__(self):
        self.outcome_collector = OutcomeCollector()
        self.feedback_manager = FeedbackManager()
        self.experience_repository = ExperienceRepository()
        self.pattern_discovery = PatternDiscovery()
        self.knowledge_updater = KnowledgeUpdater()
        self.confidence_trainer = ConfidenceTrainer()
        self.rule_optimizer = RuleOptimizer()
        self.pipeline = ContinuousLearningPipeline()
    
    async def learn(
        self,
        decision: ClinicalDecision,
        outcome: Outcome,
        feedback: Feedback | None,
    ) -> LearningPackage:
        """Process learning from decision outcome."""
        
        # 1. Collect outcome
        collected_outcome = await self.outcome_collector.collect(
            decision, outcome
        )
        
        # 2. Process feedback
        if feedback:
            processed_feedback = await self.feedback_manager.process(feedback)
        
        # 3. Store experience
        experience = await self.experience_repository.store(
            decision, collected_outcome, feedback
        )
        
        # 4. Discover patterns
        patterns = await self.pattern_discovery.discover(experience)
        
        # 5. Update knowledge
        knowledge_updates = await self.knowledge_updater.update(patterns)
        
        # 6. Train confidence
        confidence_updates = await self.confidence_trainer.train(
            experience
        )
        
        # 7. Optimize rules
        rule_suggestions = await self.rule_optimizer.optimize(
            patterns, experience
        )
        
        # 8. Generate learning package
        return LearningPackage(
            new_knowledge=knowledge_updates,
            updated_confidence=confidence_updates,
            new_patterns=patterns,
            suggested_rules=rule_suggestions,
            recommendation_scores=[],
            similar_cases=[],
            experience_records=[experience],
            learning_metrics=self._calculate_metrics(),
        )
```
