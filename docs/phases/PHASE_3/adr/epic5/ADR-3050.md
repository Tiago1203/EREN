# ADR-3050: Explainability Architecture

**Título:** Arquitectura del Motor de Explicabilidad
**Estado:** ✅ COMPLETO

**Decisor:** Architecture Team

## Contexto

El Explainability Engine convierte las recomendaciones de EREN en explicaciones comprensibles. Debe responder: "¿Por qué EREN recomienda esto?"

### Modelo de Explicación

```python
@dataclass(frozen=True)
class Explanation:
    """Complete explanation of a recommendation."""
    recommendation_id: str
    summary: str
    detailed_explanation: str
    reasoning_graph: ReasoningGraph
    evidence_tree: EvidenceTree
    decision_path: DecisionPath
    source_trace: SourceTrace
    confidence: float
    natural_language: str
    timestamp: datetime


@dataclass(frozen=True)
class ReasoningGraph:
    """Graph representation of reasoning."""
    nodes: list[GraphNode]
    edges: list[GraphEdge]
    root_node: str
    critical_path: list[str]


@dataclass(frozen=True)
class EvidenceTree:
    """Tree representation of evidence."""
    root: TreeNode
    branches: list[TreeBranch]
    annotations: dict[str, Annotation]


@dataclass(frozen=True)
class DecisionPath:
    """Path through decision space."""
    steps: list[PathStep]
    alternatives: list[AlternativePath]
    selected_path: list[str]


@dataclass(frozen=True)
class SourceTrace:
    """Trace of all sources."""
    citations: list[Citation]
    provenance: list[ProvenanceEntry]
    references: list[Reference]
```

### Motor de Explicación

```python
class ExplainabilityEngine:
    """Main explainability engine."""
    
    def __init__(self):
        self.graph_generator = ReasoningGraphGenerator()
        self.tree_builder = EvidenceTreeBuilder()
        self.path_tracer = DecisionPathTracer()
        self.source_tracer = SourceTracer()
        self.nl_explainer = NaturalLanguageExplainer()
    
    async def explain(
        self,
        recommendation: Recommendation,
        reasoning_chain: ReasoningChain,
        evidence_bundle: EvidenceBundle,
        confidence_score: ConfidenceScore,
    ) -> Explanation:
        """Generate complete explanation."""
        
        # Generate reasoning graph
        graph = await self.graph_generator.generate(reasoning_chain)
        
        # Build evidence tree
        tree = await self.tree_builder.build(evidence_bundle)
        
        # Trace decision path
        path = await self.path_tracer.trace(reasoning_chain)
        
        # Trace sources
        trace = await self.source_tracer.trace(evidence_bundle)
        
        # Generate natural language
        nl = await self.nl_explainer.generate(
            recommendation=recommendation,
            evidence_bundle=evidence_bundle,
            confidence=confidence_score,
        )
        
        return Explanation(
            recommendation_id=recommendation.id,
            summary=self._generate_summary(recommendation, confidence_score),
            detailed_explanation=nl,
            reasoning_graph=graph,
            evidence_tree=tree,
            decision_path=path,
            source_trace=trace,
            confidence=confidence_score.score,
            natural_language=nl,
            timestamp=datetime.now(),
        )
    
    def _generate_summary(
        self,
        recommendation: Recommendation,
        confidence: ConfidenceScore,
    ) -> str:
        """Generate short summary."""
        return (
            f"EREN recommends: {recommendation.action}. "
            f"Confidence: {confidence.percentage}%. "
            f"Based on {len(confidence.supporting_factors)} supporting factors."
        )
```
