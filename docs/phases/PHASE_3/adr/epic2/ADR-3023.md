# ADR-3023: Diagnostic Engine Design

**Título:** Diseño del Motor de Diagnóstico
**Estado:** ✅ COMPLETO
**Fecha:** 2026-07-21
**Decisor:** Architecture Team

---

## Contexto

El Diagnostic Engine analiza síntomas para generar diagnósticos diferenciales, identificar causas raíz y cuantificar la confianza en el diagnóstico.

---

## Decisión

### Arquitectura del Diagnostic Engine

```
DiagnosticEngine
├── SymptomAnalyzer
│   ├── PatternMatcher
│   │   # Busca patrones conocidos
│   ├── AnomalyDetector
│   │   # Detecta anomalías
│   └── SymptomGrouper
│       # Agrupa síntomas relacionados
│
├── DifferentialDiagnosis
│   ├── CandidateGenerator
│   │   # Genera candidatos diagnósticos
│   ├── EvidenceEvaluator
│   │   # Evalúa evidencia para cada candidato
│   ├── LikelihoodCalculator
│   │   # Calcula probabilidad relativa
│   └── EliminationProcessor
│       # Elimina candidatos improbables
│
├── RootCauseAnalysis
│   ├── FiveWhysAnalyzer
│   │   # Análisis de 5 Porqués
│   ├── FishboneDiagram
│   │   # Diagrama de Ishikawa
│   ├── FaultTreeAnalyzer
│   │   # Análisis de árbol de fallas
│   └── CausalChainBuilder
│       # Construye cadena causal
│
└── DiagnosticConfidence
    ├── EvidenceWeighter
    │   # Pondera evidencia por calidad
    ├── UncertaintyQuantifier
    │   # Cuantifica incertidumbre
    └── ConfidenceCalibrator
        # Calibra confianza con datos históricos
```

### Modelo de Datos

```python
@dataclass(frozen=True)
class Diagnosis:
    """Diagnóstico generado."""
    diagnosis_id: str
    condition: str
    probability: float
    confidence: ConfidenceScore
    supporting_evidence: list[Evidence]
    ruling_out_evidence: list[Evidence]
    recommendations: list[Recommendation]
    urgency: UrgencyLevel
    created_at: datetime

@dataclass(frozen=True)
class DifferentialDiagnosis:
    """Diagnóstico diferencial completo."""
    differential_id: str
    patient_context: PatientContext
    symptoms: list[Symptom]
    diagnoses: list[Diagnosis]
    most_likely: Diagnosis
    ruled_out: list[str]
    pending_investigations: list[Investigation]
    confidence_summary: ConfidenceSummary

@dataclass(frozen=True)
class RootCauseAnalysis:
    """Análisis de causa raíz."""
    rca_id: str
    problem_statement: str
    contributing_factors: list[Factor]
    root_cause: Cause
    causal_chain: list[CausalLink]
    recommendations: list[Recommendation]
    confidence: float

@dataclass(frozen=True)
class ConfidenceSummary:
    """Resumen de confianza del diagnóstico."""
    overall_confidence: float
    confidence_factors: dict[str, float]
    uncertainty_sources: list[str]
    calibration_score: float

class DiagnosticEngine:
    """Motor de diagnóstico clínico."""
    
    async def diagnose(
        self,
        symptoms: list[Symptom],
        context: ReasoningContext,
    ) -> DifferentialDiagnosis:
        """Ejecuta diagnóstico diferencial."""
        # 1. Analizar síntomas
        analyzed = await self.analyzer.analyze(symptoms, context)
        
        # 2. Generar candidatos
        candidates = await self.candidate_generator.generate(analyzed, context)
        
        # 3. Evaluar evidencia
        evaluated = await self.evidence_evaluator.evaluate(candidates, context)
        
        # 4. Calcular probabilidades
        probabilities = await self.likelihood_calculator.calculate(evaluated)
        
        # 5. Generar ranking
        ranked = self.rank_diagnoses(probabilities)
        
        # 6. Calcular confianza
        confidence = await self.confidence_calculator.calculate(ranked, context)
        
        return DifferentialDiagnosis(
            differential_id=f"dd_{datetime.now().timestamp()}",
            patient_context=context.patient,
            symptoms=symptoms,
            diagnoses=ranked,
            most_likely=ranked[0] if ranked else None,
            ruled_out=self.get_ruled_out(ranked),
            pending_investigations=self.get_pending_investigations(ranked),
            confidence_summary=confidence,
        )
    
    async def analyze_root_cause(
        self,
        problem: Problem,
        timeline: list[Event],
    ) -> RootCauseAnalysis:
        """Ejecuta análisis de causa raíz."""
        # 1. Five Whys
        why_chain = await self.five_whys.analyze(problem)
        
        # 2. Identificar factores
        factors = await self.factor_identifier.identify(problem, timeline)
        
        # 3. Construir cadena causal
        causal_chain = await self.causal_builder.build(factors)
        
        # 4. Identificar causa raíz
        root_cause = self.identify_root_cause(causal_chain)
        
        # 5. Generar recomendaciones
        recommendations = await self.recommendation_generator.generate(root_cause)
        
        return RootCauseAnalysis(
            rca_id=f"rca_{datetime.now().timestamp()}",
            problem_statement=problem.description,
            contributing_factors=factors,
            root_cause=root_cause,
            causal_chain=causal_chain,
            recommendations=recommendations,
            confidence=0.85,
        )
```

## Consecuencias

### Positivas
- ✅ Diagnóstico estructurado y reproducible
- ✅ Múltiples estrategias de análisis
- ✅ Trazabilidad completa
- ✅ Integración con Evidence Store

### Negativas
- ⚠️ Dependencia de calidad de evidencia
- ⚠️ Complejidad en causa raíz
