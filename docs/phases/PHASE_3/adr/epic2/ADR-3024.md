# ADR-3024: Causal Graph Design

**Título:** Diseño del Grafo Causal
**Estado:** ✅ COMPLETO
**Fecha:** 2026-07-21
**Decisor:** Architecture Team

---

## Contexto

El Causal Graph representa y analiza relaciones causales entre eventos, fallas y condiciones. Es fundamental para el razonamiento sobre causa raíz y predicción de efectos.

---

## Decisión

### Arquitectura del Causal Graph

```
CausalGraph
├── CausalRelations
│   ├── DirectCausation
│   │   # Causa directa A → B
│   ├── ContributingFactor
│   │   # Factor contribuyente (A + B → C)
│   ├── EnablingCondition
│   │   # Condición habilitadora
│   └── TemporalRelationship
│       # Relación temporal (A occurs_before B)
│
├── CausalInference
│   ├── DoCalculus
│   │   # Intervención (do(A) → efecto en B)
│   ├── Counterfactual
│   │   # Contrafácticos (qué pasaría si...)
│   ├── MediationAnalysis
│   │   # Análisis de mediación (A → M → B)
│   └── EffectDecomposition
│       # Descomposición de efectos
│
├── EvidenceLinking
│   ├── EvidenceToCauseMapper
│   │   # Mapea evidencia a causas
│   ├── ConfidencePropagation
│   │   # Propaga confianza por el grafo

```

### Modelo de Datos

```python
@dataclass(frozen=True)
class CausalNode:
    """Nodo en el grafo causal."""
    node_id: str
    node_type: CausalNodeType  # event, factor, condition
    description: str
    probability: float
    evidence: list[Evidence]
    properties: dict

@dataclass(frozen=True)
class CausalEdge:
    """Arista causal."""
    edge_id: str
    source_id: str
    target_id: str
    causal_type: CausalType  # direct, contributing, enabling
    strength: float  # 0.0 - 1.0
    temporal_relation: TemporalRelation | None
    conditions: list[Condition]
    evidence: list[Evidence]

@dataclass(frozen=True)
class CausalPath:
    """Camino causal entre nodos."""
    path_id: str
    nodes: list[CausalNode]
    edges: list[CausalEdge]
    total_strength: float
    probability: float

class CausalGraph:
    """Grafo de relaciones causales."""
    
    def __init__(self):
        self._nodes: dict[str, CausalNode] = {}
        self._edges: dict[str, list[CausalEdge]] = defaultdict(list)
        self._reverse_edges: dict[str, list[CausalEdge]] = defaultdict(list)
    
    async def add_causal_relation(
        self,
        source: CausalNode,
        target: CausalNode,
        causal_type: CausalType,
        strength: float,
    ) -> CausalEdge:
        """Agrega relación causal."""
        edge = CausalEdge(
            edge_id=f"ce_{len(self._edges)}",
            source_id=source.node_id,
            target_id=target.node_id,
            causal_type=causal_type,
            strength=strength,
            temporal_relation=None,
            conditions=[],
            evidence=[],
        )
        self._edges[source.node_id].append(edge)
        self._reverse_edges[target.node_id].append(edge)
        return edge
    
    async def infer_effects(
        self,
        cause_node_id: str,
    ) -> list[CausalNode]:
        """Infiere efectos de una causa."""
        effects = []
        visited = set()
        queue = deque([cause_node_id])
        
        while queue:
            current = queue.popleft()
            if current in visited:
                continue
            visited.add(current)
            
            for edge in self._edges.get(current, []):
                target = self._nodes.get(edge.target_id)
                if target:
                    effects.append(target)
                    queue.append(edge.target_id)
        
        return effects
    
    async def infer_causes(
        self,
        effect_node_id: str,
    ) -> list[CausalNode]:
        """Infiere causas de un efecto."""
        causes = []
        for edge in self._reverse_edges.get(effect_node_id, []):
            cause = self._nodes.get(edge.source_id)
            if cause:
                causes.append(cause)
        return causes
    
    async def calculate_causal_strength(
        self,
        path: CausalPath,
    ) -> float:
        """Calcula fuerza causal de un camino."""
        if not path.edges:
            return 0.0
        
        total = 1.0
        for edge in path.edges:
            total *= edge.strength
        
        return total
    
    async def propagate_confidence(
        self,
        node_id: str,
        confidence: float,
    ) -> dict[str, float]:
        """Propaga confianza por el grafo."""
        confidences = {node_id: confidence}
        queue = deque([(node_id, confidence)])
        visited = {node_id}
        
        while queue:
            current, conf = queue.popleft()
            
            for edge in self._edges.get(current, []):
                if edge.target_id not in visited:
                    new_conf = conf * edge.strength
                    confidences[edge.target_id] = new_conf
                    queue.append((edge.target_id, new_conf))
                    visited.add(edge.target_id)
        
        return confidences
```

### Tipos de Relaciones Causales

```python
class CausalType(Enum):
    """Tipos de relaciones causales."""
    DIRECT = "direct"                    # A causa B directamente
    CONTRIBUTING = "contributing"      # A contribuye a B (necesita otros factores)
    ENABLING = "enabling"              # A habilita B (B no ocurre sin A)
    PREVENTING = "preventing"          # A previene B
    MITIGATING = "mitigating"          # A mitiga B
    TRIGGERING = "triggering"         # A dispara B
    MODIFYING = "modifying"            # A modifica la probabilidad de B


class CausalNodeType(Enum):
    """Tipos de nodos causales."""
    EVENT = "event"                    # Evento observado
    FACTOR = "factor"                  # Factor contribuyente
    CONDITION = "condition"            # Condición de fondo
    MECHANISM = "mechanism"           # Mecanismo
    OUTCOME = "outcome"               # Resultado


class TemporalRelation(Enum):
    """Relaciones temporales."""
    OCCURS_BEFORE = "occurs_before"
    OCCURS_AFTER = "occurs_after"
    SIMULTANEOUS = "simultaneous"
    OVERLAPPING = "overlapping"
```

## Consecuencias

### Positivas
- ✅ Representación estructurada de causalidad
- ✅ Inferencia de causas y efectos
- ✅ Propagación de confianza
- ✅ Soporte para análisis de causa raíz

### Negativas
- ⚠️ Complejidad en grafos grandes
- ⚠️ Ciclos causales (mitigados con validación)
