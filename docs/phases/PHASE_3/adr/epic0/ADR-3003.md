# ADR-3003: Safety Model Design

**Título:** Diseño de Modelos de Seguridad Clínica
**Estado:** ✅ ACEPTADO
**Fecha:** 2026-07-21
**Decisor:** Architecture Team

---

## Contexto

En entornos clínicos, la seguridad del paciente es primordial. Necesitamos:

1. Clasificar la severidad de problemas de seguridad
2. Detectar contraindicaciones y advertencias
3. Generar alertas apropiadas
4. Rastrear compliance con safety checks

---

## Decisión

### Niveles de Seguridad

```python
class SafetyLevel(Enum):
    """Nivel de seguridad clínica."""
    CRITICAL = "critical"      # Pone en riesgo la vida
    HIGH = "high"              # Daño serio potencial
    MEDIUM = "medium"          # Daño moderado potencial
    LOW = "low"                # Informacional
    INFORMATIONAL = "informational"  # Para documentación
```

### Safety Check

```python
@dataclass(frozen=True)
class SafetyCheck:
    """Verificación de seguridad."""
    check_id: str
    check_type: SafetyCheckType
    description: str
    level: SafetyLevel
    rule_id: str               # Referencia a regla violated
    patient_context: dict
    recommendation: str
    requires_acknowledgment: bool
```

### Clinical Warning

```python
@dataclass(frozen=True)
class ClinicalWarning:
    """Advertencia clínica."""
    warning_id: str
    level: SafetyLevel
    title: str
    message: str
    affected_entities: list[str]
    actions_required: list[str]
    escalation_path: str | None
    created_at: datetime
    acknowledged: bool = False
```

---

## Tipos de Safety Checks

```python
class SafetyCheckType(Enum):
    """Tipos de verificación de seguridad."""
    DRUG_INTERACTION = "drug_interaction"
    ALLERGY_CHECK = "allergy_check"
    CONTRAINDICATION = "contraindication"
    DOSE_RANGE = "dose_range"
    DUPLICATE_THERAPY = "duplicate_therapy"
    AGE_APPROPRIATENESS = "age_appropriateness"
    PREGNANCY_CHECK = "pregnancy_check"
    LAB_VALUE = "lab_value"
    VITAL_SIGN = "vital_sign"
    CUSTOM = "custom"
```

## Consecuencias

### Positivas
- ✅ Clasificación clara de severidades
- ✅ Escalamiento automático según nivel
- ✅ Rastreo de acknowledgment

### Negativas
- ⚠️ Alertas excesivas pueden causar fatigue
