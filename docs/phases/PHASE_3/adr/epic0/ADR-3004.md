# ADR-3004: Confidence Interface Design

**Título:** Diseño de Interfaces de Confianza
**Estado:** ✅ ACEPTADO
**Fecha:** 2026-07-21
**Decisor:** Architecture Team

---

## Contexto

La confianza en las recomendaciones clínicas debe ser cuantificable y comunicada claramente. Necesitamos:

1. Calcular scores de confianza basados en evidencia
2. Propagar incertidumbre a través del razonamiento
3. Comunicar niveles de confianza a usuarios
4. Soportar calibración de modelos

---

## Decisión

### Confidence Score

```python
@dataclass(frozen=True)
class ConfidenceScore:
    """Score de confianza."""
    value: float                  # 0.0 - 1.0
    confidence_level: ConfidenceLevel
    factors: list[ConfidenceFactor]
    calibration_data: CalibrationData | None
    estimated_accuracy: float | None
```

### Niveles de Confianza

```python
class ConfidenceLevel(Enum):
    """Nivel de confianza."""
    VERY_HIGH = "very_high"      # > 0.95
    HIGH = "high"                # 0.85 - 0.95
    MODERATE = "moderate"        # 0.70 - 0.85
    LOW = "low"                  # 0.50 - 0.70
    VERY_LOW = "very_low"        # < 0.50
    UNCERTAIN = "uncertain"      # No puede determinarse
```

## Factores de Confianza

```python
@dataclass(frozen=True)
class ConfidenceFactor:
    """Factor que afecta la confianza."""
    factor_type: ConfidenceFactorType
    contribution: float           # Contribución al score final
    description: str
    source_evidence: str
```

### Tipos de Factores

```python
class ConfidenceFactorType(Enum):
    """Tipos de factores de confianza."""
    EVIDENCE_QUALITY = "evidence_quality"
    EVIDENCE_QUANTITY = "evidence_quantity"
    CONSISTENCY = "consistency"
    NOVELTY = "novelty"
    PATIENT_MATCH = "patient_match"
    MODEL_CONFIDENCE = "model_confidence"
    SOURCE_RELIABILITY = "source_reliability"
```
