# ADR-3014: Evidence Store Architecture

**Título:** Arquitectura del Evidence Store
**Estado:** 🚧 IN PROGRESS
**Fecha:** 2026-07-21
**Decisor:** Architecture Team

---

## Contexto

El sistema necesita almacenar y recuperar evidencia de múltiples fuentes para alimentar el razonamiento clínico:

1. **Literature Evidence**: PubMed, ClinicalTrials.gov
2. **Standard Evidence**: ISO, IEC, AAMI documents
3. **Regulatory Evidence**: FDA reports, ECRI, PM (Post-Market)
4. **Clinical Guidelines**: WHO, clinical practice guidelines
5. **Real-World Evidence**: Device data, incident reports

---

## Decisión

### Arquitectura del Evidence Store

```
EvidenceStore
├── LiteratureSource
│   ├── PubMedIntegration
│   ├── ClinicalTrialsIntegration
│   └── CochraneLibraryIntegration
├── StandardSource
│   ├── ISOStandardsIntegration
│   ├── IECStandardsIntegration
│   └── AAMIIntegration
├── RegulatorySource
│   ├── FDAIntegration (MAUDE, recalls)
│   ├── ECRIIntegration
│   └── MDRReports
├── GuidelineSource
│   ├── WHOIntegration
│   ├── NICEIntegration
│   └── InstitutionalGuidelines
└── RealWorldSource
    ├── IncidentReports
    ├── MaintenanceLogs
    └── DeviceTelemetry
```

### Modelo de Datos

```python
@dataclass(frozen=True)
class Evidence:
    """Evidencia almacenada."""
    evidence_id: str
    source_type: EvidenceSourceType
    source_id: str
    title: str
    abstract: str | None
    full_text: str | None
    url: str
    authors: list[str] | None
    publication_date: date | None
    evidence_level: EvidenceLevel
    content_hash: str
    indexed_at: datetime
    expires_at: datetime | None
    metadata: dict

@dataclass(frozen=True)
class EvidenceRetrieval:
    """Resultado de recuperación de evidencia."""
    retrieval_id: str
    query: str
    evidence_found: list[Evidence]
    relevance_scores: dict[str, float]
    retrieval_time_ms: int
    total_available: int
    filters_applied: dict

@dataclass(frozen=True)
class EvidenceChain:
    """Cadena de evidencia para una conclusión."""
    chain_id: str
    conclusion_id: str
    supporting_evidence: list[EvidenceWithRelevance]
    contradicting_evidence: list[EvidenceWithRelevance]
    overall_evidence_level: EvidenceLevel
    consistency_score: float
    confidence_modifier: float

@dataclass(frozen=True)
class EvidenceWithRelevance:
    """Evidencia con puntaje de relevancia."""
    evidence: Evidence
    relevance_score: float
    relevance_reasons: list[str]
    applicability_to_case: ApplicabilityAssessment

class EvidenceStore:
    """Almacén de evidencia."""
    
    async def add_evidence(
        self,
        evidence: Evidence,
    ) -> str:
        """Agrega evidencia al store."""
        ...
    
    async def retrieve_evidence(
        self,
        query: EvidenceQuery,
        filters: EvidenceFilters | None = None,
        limit: int = 20,
    ) -> EvidenceRetrieval:
        """Recupera evidencia relevante."""
        ...
    
    async def get_evidence_by_source(
        self,
        source_type: EvidenceSourceType,
        source_id: str,
    ) -> Evidence | None:
        """Obtiene evidencia por fuente."""
        ...
    
    async def build_evidence_chain(
        self,

        ...

        self,

        """Actualiza fecha de expiración."""
```

### Tipos de Fuentes de Evidencia

```python
class EvidenceSourceType(Enum):
    """Tipos de fuentes de evidencia."""
    # Literatura científica
    PUBMED = "pubmed"
    CLINICAL_TRIALS = "clinical_trials"
    COCHRANE = "cochrane"
    EMBASE = "embase"
    
    # Estándares
    ISO_STANDARD = "iso_standard"
    IEC_STANDARD = "iec_standard"
    AAMI_GUIDELINE = "aami_guideline"
    
    # Regulatorio
    FDA_MAUDE = "fda_maude"
    FDA_RECALL = "fda_recall"
    ECRI_REPORT = "ecri_report"
    EU_MDR = "eu_mdr"
    
    # Guías clínicas
    WHO_GUIDELINE = "who_guideline"
    NICE_GUIDELINE = "nice_guideline"
    INSTITUTIONAL_GUIDELINE = "institutional_guideline"
    
    # Evidencia del mundo real
    INCIDENT_REPORT = "incident_report"
    MAINTENANCE_LOG = "maintenance_log"
    DEVICE_TELEMETRY = "device_telemetry"
    REAL_WORLD_EVIDENCE = "real_world_evidence"

class EvidenceLevel(Enum):
    """Nivel de evidencia."""
    LEVEL_1_SYSTEMATIC = "level_1_systematic"  # Revisión sistemática
    LEVEL_1_RCT = "level_1_rct"                # ECA de alta calidad
    LEVEL_2_COHORT = "level_2_cohort"           # Estudios de cohorte
    LEVEL_3_CASE_CONTROL = "level_3_case_control"
    LEVEL_4_CASE_SERIES = "level_4_case_series"
    LEVEL_5_EXPERT = "level_5_expert"          # Opinión de expertos
    LEVEL_6_MECHANISTIC = "level_6_mechanistic" # Investigación básica
    LEVEL_7_REAL_WORLD = "level_7_real_world"  # Evidencia del mundo real
```

### Filtros de Búsqueda

```python
@dataclass(frozen=True)
class EvidenceFilters:
    """Filtros para búsqueda de evidencia."""
    source_types: list[EvidenceSourceType] | None = None
    evidence_levels: list[EvidenceLevel] | None = None
    date_from: date | None = None
    date_to: date | None = None
    relevance_threshold: float | None = None
    device_categories: list[str] | None = None
    standards: list[str] | None = None
    keywords: list[str] | None = None

@dataclass(frozen=True)
class EvidenceQuery:
    """Query de búsqueda de evidencia."""
    query_text: str
    query_type: QueryType
    context: dict | None = None

class QueryType(Enum):
    """Tipos de query."""
    SEMANTIC = "semantic"          # Búsqueda semántica
    KEYWORD = "keyword"            # Búsqueda por palabras clave
    VECTOR = "vector"              # Búsqueda por embeddings
    HYBRID = "hybrid"              # Combinación
```

## Consecuencias

### Positivas
- ✅ Acceso centralizado a evidencia
- ✅ Búsqueda semántica y por palabras clave
- ✅ Cache para performance
- ✅ Versionamiento y expiración de evidencia

### Negativas
- ⚠️ Costo de almacenamiento puede crecer
- ⚠️ Actualización de fuentes externas
- ⚠️ Validación de calidad de evidencia
