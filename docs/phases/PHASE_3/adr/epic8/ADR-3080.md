# ADR-3080: Clinical Validation Architecture

**Título:** Arquitectura de Validación Clínica
**Estado:** ✅ COMPLETO
**Fecha:** 2026-07-21

## Contexto

Clinical Validation es el punto final de validación antes de emitir cualquier respuesta. Valida evidencia, confianza, seguridad, reglas y consistencia.

### Modelo de Validación

```python
class ValidationDecision(Enum):
    """Decision outcomes."""
    APPROVED = "approved"
    REJECTED = "rejected"
    REQUIRES_CORRECTION = "requires_correction"
    ESCALATE = "escalate"


@dataclass
class ValidationResult:
    """Result of clinical validation."""
    decision: ValidationDecision
    is_valid: bool
    evidence_valid: bool
    confidence_valid: bool
    safety_valid: bool
    rules_valid: bool
    consistency_valid: bool
    clinical_valid: bool
    rejection_reasons: list[str]
    correction_suggestions: list[str]
    confidence_adjustment: float | None


class ClinicalValidationEngine:
    """Main clinical validation engine."""
    
    def __init__(self):
        self.evidence_validator = EvidenceValidator()
        self.confidence_checker = ConfidenceChecker()
        self.safety_gate = SafetyGate()
        self.rules_checker = RulesChecker()
        self.consistency_checker = ConsistencyChecker()
        self.appropriateness_validator = ClinicalAppropriatenessValidator()
    
    async def validate(
        self,
        recommendation: Recommendation,
        context: dict,
        safety_result: SafetyResult,
        rules_result: ValidationResult,
        confidence_score: ConfidenceScore,
    ) -> ValidationResult:
        """Validate recommendation through complete pipeline."""
        
        rejection_reasons = []
        corrections = []
        
        # 1. Evidence Validation
        evidence_valid = await self.evidence_validator.validate(
            recommendation, context
        )
        if not evidence_valid.is_sufficient:
            rejection_reasons.append(
                f"Insufficient evidence: {evidence_valid.reason}"
            )
        
        # 2. Confidence Check
        confidence_valid = self.confidence_checker.check(
            confidence_score
        )
        if not confidence_valid.is_acceptable:
            rejection_reasons.append(
                f"Confidence too low: {confidence_score.score:.0%}"
            )
        
        # 3. Safety Gate
        if safety_result.decision == SafetyDecision.BLOCK:
            rejection_reasons.append("Safety validation failed")
        
        # 4. Rules Check
        if not rules_result.is_compliant:
            rejection_reasons.append(
                f"Rules violation: {rules_result.violations[0].description}"
            )
        
        # 5. Consistency Check
        consistency_valid = await self.consistency_checker.check(
            recommendation, context
        )
        if not consistency_valid.is_consistent:
            corrections.extend(consistency_valid.suggestions)
        
        # 6. Clinical Appropriateness
        clinical_valid = await self.appropriateness_validator.validate(
            recommendation, context
        )
        if not clinical_valid.is_appropriate:
            rejection_reasons.append(
                f"Clinical appropriateness: {clinical_valid.reason}"
            )
        
        # Determine final decision
        decision = self._determine_decision(
            rejection_reasons, corrections
        )
        
        return ValidationResult(
            decision=decision,
            is_valid=decision == ValidationDecision.APPROVED,
            evidence_valid=evidence_valid.is_sufficient,
            confidence_valid=confidence_valid.is_acceptable,
            safety_valid=safety_result.decision != SafetyDecision.BLOCK,
            rules_valid=rules_result.is_compliant,
            consistency_valid=consistency_valid.is_consistent,
            clinical_valid=clinical_valid.is_appropriate,
            rejection_reasons=rejection_reasons,
            correction_suggestions=corrections,
            confidence_adjustment=consistency_valid.adjustment,
        )
```

## Consecuencias

### Positivas
- ✅ Validación completa antes de responder
- ✅ Autocorrección cuando es posible
- ✅ Rechazo cuando es necesario

### Negativas
- ⚠️ Puede rechazar respuestas válidas
- ⚠️ Complejidad en pipeline
