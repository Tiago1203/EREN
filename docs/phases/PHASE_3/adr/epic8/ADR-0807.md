# ADR-0807: Monitoring Stack

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Comprehensive monitoring ensures system health and enables rapid incident response.

## Decision

We will implement **Prometheus + Grafana Stack**:

### Components

| Component | Purpose | Retention |
|-----------|---------|-----------|
| Prometheus | Metrics collection | 30 days |
| Grafana | Visualization | - |
| AlertManager | Alert routing | - |
| Loki | Log aggregation | 90 days |
| Tempo | Distributed tracing | 30 days |

### Metrics Collected

```yaml
metrics:
  infrastructure:
    - cpu_usage
    - memory_usage
    - disk_io
    - network_io
  application:
    - request_rate
    - error_rate
    - latency_p50_p95_p99
    - active_connections
  business:
    - devices_monitored
    - incidents_created
    - ai_recommendations
```

### Dashboards

1. **System Overview** — Infrastructure health
2. **Application Performance** — API metrics
3. **Business KPIs** — Device/incident counts
4. **AI Performance** — Recommendation accuracy
