# ADR-3060: Biomedical Rules Architecture

**Título:** Arquitectura del Motor de Reglas Biomédicas
**Estado:** ✅ COMPLETO
**Fecha:** 2026-07-21
**Decisor:** Architecture Team

---

## Contexto

El Biomedical Rules Engine valida recomendaciones contra estándares regulatorios y reglas clínicas. **NO usa IA** - es determinístico.

---

## Decisión

### Arquitectura

```
BiomedicalRulesEngine (extends EPIC 3 RulesEngine)
├── IECValidator
│   ├── IEC60601Validator
│   ├── IEC62304Validator
│   └── IEC62353Validator
├── ISOValidator
│   ├── ISO13485Validator
│   ├── ISO14971Validator
│   └── ISO15223Validator
├── AAMIValidator
├── ClinicalRulesEngine
│   ├── AlarmRules
│   ├── DosingRules
│   └── SafetyRules
└── EngineeringRulesEngine
    ├── MaintenanceRules
    ├── CalibrationRules
    └── ElectricalRules
```

### Modelo de Validación

```python
@dataclass
class ValidationResult:
    """Result of rules validation."""
    is_compliant: bool
    violations: list[Violation]
    warnings: list[Warning]
    recommendations: list[str]
    risk_level: RiskLevel
    validation_time_ms: int


@dataclass
class Violation:
    """Rule violation."""
    rule_id: str
    standard: str
    clause: str
    description: str
    severity: Severity
    current_value: Any
    required_value: Any


class BiomedicalRulesEngine:
    """Main rules engine for biomedical validation."""
    
    def __init__(self):
        self.iec_validator = IECValidator()
        self.iso_validator = ISOValidator()
        self.aami_validator = AAMIValidator()
        self.clinical_rules = ClinicalRulesEngine()
        self.engineering_rules = EngineeringRulesEngine()
    
    async def validate(
        self,
        equipment: Equipment,
        recommendation: Recommendation,
        context: dict,
    ) -> ValidationResult:
        """Validate recommendation against all rules."""
        
        violations = []
        warnings = []
        recommendations = []
        
        # IEC Validation
        iec_result = await self.iec_validator.validate(equipment, context)
        violations.extend(iec_result.violations)
        warnings.extend(iec_result.warnings)
        
        # ISO Validation
        iso_result = await self.iso_validator.validate(equipment, context)
        violations.extend(iso_result.violations)
        
        # Clinical Rules
        clinical_result = await self.clinical_rules.evaluate(context)
        violations.extend(clinical_result.violations)
        
        # Engineering Rules
        eng_result = await self.engineering_rules.evaluate(context)
        violations.extend(eng_result.violations)
        
        # Calculate risk level
        risk_level = self._calculate_risk_level(violations)
        
        # Generate recommendations
        recommendations = self._generate_recommendations(violations)
        
        return ValidationResult(
            is_compliant=len(violations) == 0,
            violations=violations,
            warnings=warnings,
            recommendations=recommendations,
            risk_level=risk_level,
            validation_time_ms=0,
        )
```
