# ADR-0709: Authentication & Authorization

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Secure access control for all EREN users with role-based permissions.

## Decision

We will implement **OAuth 2.0 + JWT**:

### Auth Flow

```
┌─────────┐     ┌──────────────┐     ┌─────────┐
│  User   │ →  │   Azure AD    │ →  │  EREN   │
│  Browser│    │ (SAML/OIDC)  │    │  API    │
└─────────┘     └──────────────┘     └─────────┘
```

### Roles

| Role | Permissions |
|------|-------------|
| Admin | Full access |
| Engineer | CRUD devices, incidents |
| Viewer | Read-only |
| Manager | Reports, analytics |

### Implementation

```typescript
interface UserClaims {
  sub: string;        // User ID
  email: string;      // Email
  roles: string[];    // Role array
  hospital: string;   // Hospital ID
}
```
