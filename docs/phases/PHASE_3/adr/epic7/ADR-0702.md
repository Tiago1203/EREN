# ADR-0702: Dashboard Design

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Dashboards provide real-time visibility into hospital operations and device status.

## Decision

We will implement role-based dashboards:

### Dashboard Types

| Dashboard | Audience | Metrics |
|-----------|----------|---------|
| Executive | Leadership | KPIs, trends |
| Operations | Managers | Capacity, incidents |
| Clinical | Engineers | Devices, maintenance |
| AI Insights | All | CDSS recommendations |

### Widgets

1. **Device Status Grid** — Real-time monitoring
2. **Incident Timeline** — Recent alerts
3. **Capacity Heatmap** — Bed utilization
4. **AI Recommendations** — Prioritized actions
5. **Maintenance Calendar** — Scheduled tasks
