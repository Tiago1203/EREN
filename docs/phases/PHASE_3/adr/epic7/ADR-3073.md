# ADR-3073: Critical Alerts Design

**Título:** Diseño de Alertas Críticas
**Estado:** ✅ COMPLETO
**Fecha:** 2026-07-21
**Decisor:** Architecture Team

---

## Contexto

Sistema de alertas críticas para escalar situaciones de riesgo.

---

## Decisión

### Niveles de Escalación

```
Alert Levels
├── Level 1: System Warning
│   ├── Description: Minor safety concern
│   ├── Action: Log and continue
│   └── Recipient: System log only
│
├── Level 2: Supervisor Notification
│   ├── Description: Moderate safety concern
│   ├── Action: Notify supervisor
│   └── Recipient: Department supervisor
│
├── Level 3: Management Escalation
│   ├── Description: Significant safety concern
│   ├── Action: Page manager, document incident
│   └── Recipient: Department manager + safety officer
│
└── Level 4: Emergency Protocol
    ├── Description: Immediate danger to patients
    ├── Action: Emergency response, halt operations
    └── Recipient: Emergency response team + administration
```

### Sistema de Alertas

```python
class AlertLevel(Enum):
    """Alert escalation levels."""
    LEVEL_1_SYSTEM = "level_1_system"
    LEVEL_2_SUPERVISOR = "level_2_supervisor"
    LEVEL_3_MANAGEMENT = "level_3_management"
    LEVEL_4_EMERGENCY = "level_4_emergency"


@dataclass
class CriticalAlert:
    """Critical alert definition."""
    alert_id: str
    level: AlertLevel
    title: str
    description: str
    affected_equipment: list[str]
    affected_patients: list[str]
    recommended_action: str
    escalation_path: list[str]
    timestamp: datetime


class CriticalAlertSystem:
    """System for generating and managing critical alerts."""
    
    def __init__(self):
        self.notification_service = NotificationService()
        self.audit_logger = AuditLogger()
    
    async def should_alert(
        self,
        safety_result: SafetyResult,
    ) -> bool:
        """Determine if alert should be triggered."""
        # Always alert for critical blockers
        if any(b.severity == Severity.CRITICAL for b in safety_result.blockers):
            return True
        
        # Alert for high risk score
        if safety_result.risk_score >= 0.8:
            return True
        
        # Alert for human escalation required
        if safety_result.requires_human_review:
            return True
        
        return False
    
    async def generate_alert(
        self,
        safety_result: SafetyResult,
        context: dict,
    ) -> CriticalAlert:
        """Generate critical alert."""
        
        # Determine alert level
        level = self._determine_alert_level(safety_result)
        
        # Build alert
        alert = CriticalAlert(
            alert_id=f"alert_{datetime.now().timestamp()}",
            level=level,
            title=self._generate_title(safety_result),
            description=self._generate_description(safety_result),
            affected_equipment=[context.get("equipment_id", "unknown")],
            affected_patients=context.get("affected_patients", []),
            recommended_action=self._generate_action(safety_result),
            escalation_path=self._get_escalation_path(level),
            timestamp=datetime.now(),
        )
        
        # Send notification
        await self._send_notification(alert, level)
        
        # Log for audit
        await self._log_alert(alert)
        
        return alert
    
    def _determine_alert_level(
        self,
        result: SafetyResult,
    ) -> AlertLevel:
        """Determine alert level based on severity."""
        # Check for critical blockers
        if any(b.severity == Severity.CRITICAL for b in result.blockers):
            return AlertLevel.LEVEL_4_EMERGENCY
        
        # Check for high risk
        if result.risk_score >= 0.9:
            return AlertLevel.LEVEL_4_EMERGENCY
        elif result.risk_score >= 0.7:
            return AlertLevel.LEVEL_3_MANAGEMENT
        elif result.risk_score >= 0.5:
            return AlertLevel.LEVEL_2_SUPERVISOR
        else:
            return AlertLevel.LEVEL_1_SYSTEM
    
    async def _send_notification(
        self,
        alert: CriticalAlert,
        level: AlertLevel,
    ):
        """Send alert notification."""
        
        if level == AlertLevel.LEVEL_4_EMERGENCY:
            # Emergency: page everyone
            await self.notification_service.emergency_page(
                message=str(alert),
                recipients=alert.escalation_path,
            )
            await self.notification_service.sms_alert(
                message=f"EMERGENCY: {alert.title}",
                recipients=alert.escalation_path,
            )
        
        elif level == AlertLevel.LEVEL_3_MANAGEMENT:
            # Page manager, email safety officer
            await self.notification_service.page(
                message=str(alert),
                recipients=alert.escalation_path,
            )
            await self.notification_service.email(
                subject=f"Safety Alert: {alert.title}",
                body=str(alert),
                recipients=alert.escalation_path,
            )
        
        elif level == AlertLevel.LEVEL_2_SUPERVISOR:
            # Notify supervisor
            await self.notification_service.email(
                subject=f"Safety Notice: {alert.title}",
                body=str(alert),
                recipients=alert.escalation_path,
            )
        
        # Level 1: System log only (no immediate notification)
    
    async def _log_alert(self, alert: CriticalAlert):
        """Log alert for audit trail."""
        await self.audit_logger.log(
            event_type="CRITICAL_ALERT",
            event_data=alert.__dict__,
            timestamp=alert.timestamp,
        )
```

## Consecuencias

### Positivas
- ✅ Escalación apropiada
- ✅ Auditoría completa
- ✅ Respuesta rápida a emergencias

### Negativas
- ⚠️ Alerta fatigue si sobre-usado
- ⚠️ Requiere integración con notificación
