## Contexto

Integración de regulaciones internacionales en el Rules Engine:
- IEC (International Electrotechnical Commission)
- ISO (International Organization for Standardization)
- FDA (US Food and Drug Administration)
- AAMI (Association for the Advancement of Medical Instrumentation)
- NFPA (National Fire Protection Association)

### Compliance Checker

```python
class ComplianceChecker:
    """Verifica compliance con regulaciones."""
    
    async def check_compliance(
        self,
        equipment_id: str,
        regulations: list[str],
    ) -> ComplianceReport:
        """Genera reporte de compliance."""
        equipment = await self.get_equipment(equipment_id)
        context = self._build_context(equipment)
        
        rules = self._get_rules_for_regulations(regulations)
        matches = await self.rules_engine.evaluate(context, rules)
        
        # Calculate compliance status
        critical_violations = [m for m in matches if m.priority == Priority.CRITICAL and not m.matched]
        high_violations = [m for m in matches if m.priority == Priority.HIGH and not m.matched]
        
        if critical_violations:
            status = ComplianceStatus.NON_COMPLIANT
        elif high_violations:
            status = ComplianceStatus.WARNING
        else:
            status = ComplianceStatus.COMPLIANT
        
        return ComplianceReport(
            equipment_id=equipment_id,
            regulations_checked=regulations,
            status=status,
            violations=[*critical_violations, *high_violations],
            recommendations=self._get_recommendations(matches),
        )
```
