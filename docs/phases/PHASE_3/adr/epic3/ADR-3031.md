# ADR-3031: Evidence Bundle Design

**Título:** Diseño del Evidence Bundle
**Estado:** ✅ COMPLETO
**Fecha:** 2026-07-21
**Decisor:** Architecture Team

---

## Contexto

El Evidence Bundle agrupa toda la evidencia relacionada con un razonamiento específico, separando evidencia a favor y en contra, y calculando una puntuación de confianza general.

---

## Decisión

### Arquitectura del Evidence Bundle

### Modelo de Datos

```python
@dataclass(frozen=True)
class EvidenceBundle:
    """Bundle de evidencia para un razonamiento."""
    bundle_id: str
    hypothesis_id: str
    supporting_evidence: list[EvidenceItem]
    contradicting_evidence: list[EvidenceItem]
    summary: EvidenceSummary
    rule_matches: list[RuleMatch]
    created_at: datetime
    
    @property
    def overall_confidence(self) -> float:
        """Calcula confianza general."""
        supporting_strength = sum(
            e.quality_score * e.relevance_score 
            for e in self.supporting_evidence
        ) / max(len(self.supporting_evidence), 1)
        
        contradicting_strength = sum(
            e.quality_score * e.relevance_score 
            for e in self.contradicting_evidence
        ) / max(len(self.contradicting_evidence), 1)
        
        # Balance calculation
        total = supporting_strength + contradicting_strength
        if total == 0:
            return 0.5
        
        return supporting_strength / total

@dataclass(frozen=True)
class EvidenceSummary:
    """Resumen de la evidencia."""
    overall_confidence: float
    evidence_count: int
    supporting_count: int
    contradicting_count: int
    highest_quality_evidence: EvidenceItem | None
    sources: list[EvidenceSource]
    recommendation: str

@dataclass(frozen=True)
class RuleMatch:
    """Coincidencia con una regla."""
    rule_id: str
    rule_name: str
    matched: bool
    conditions_met: list[str]
    conditions_not_met: list[str]
    action: str
    priority: Priority
```

### Generación de Evidence Bundle

```python
class EvidenceBundleGenerator:
    """Genera Evidence Bundles."""
    
    async def generate(
        self,
        hypothesis: Hypothesis,
        evidence_items: list[EvidenceItem],
        rules: list[Rule],
    ) -> EvidenceBundle:
        """Genera un bundle para una hipótesis."""
        # Separate supporting and contradicting
        supporting = []
        contradicting = []
        
        for evidence in evidence_items:
            if self._supports_hypothesis(evidence, hypothesis):
                supporting.append(evidence)
            else:
                contradicting.append(evidence)
        
        # Calculate summary
        summary = self._calculate_summary(
            supporting, contradicting, hypothesis
        )
        
        # Match rules
        rule_matches = self._match_rules(hypothesis, rules)
        
        return EvidenceBundle(
            bundle_id=f"bundle_{hypothesis.hypothesis_id}",
            hypothesis_id=hypothesis.hypothesis_id,
            supporting_evidence=supporting,
            contradicting_evidence=contradicting,
            summary=summary,
            rule_matches=rule_matches,
            created_at=datetime.now(),
        )
    
    def _supports_hypothesis(
        self,
        evidence: EvidenceItem,
        hypothesis: Hypothesis,
    ) -> bool:
        """Determina si evidencia apoya la hipótesis."""
        # Simple keyword matching for demonstration
        hypothesis_keywords = set(hypothesis.name.lower().split())
        evidence_keywords = set(evidence.content.lower().split())
        
        overlap = hypothesis_keywords & evidence_keywords
        return len(overlap) > 0
```

## Consecuencias

### Positivas
- ✅ Estructura clara para evidencia
- ✅ Separación de evidencia a favor/en contra
- ✅ Cálculo de confianza ponderado

### Negativas
- ⚠️ Complejidad en determinación de apoyo/contradicción
