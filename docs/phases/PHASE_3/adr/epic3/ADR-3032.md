# ADR-3032: Biomedical Rules Engine Design

**Título:** Diseño del Motor de Reglas Biomédicas
**Estado:** ✅ COMPLETO
**Fecha:** 2026-07-21
**Decisor:** Architecture Team

---

## Contexto

El Biomedical Rules Engine implementa un motor de reglas estilo Drools para validar recomendaciones clínicas y de ingeniería contra estándares regulatorios.

**NO usa IA**. Usa reglas determinísticas.

---

### Modelo de Reglas

```python
@dataclass(frozen=True)
class Rule:
    """Regla biomédica."""
    rule_id: str
    name: str
    description: str
    category: RuleCategory
    priority: Priority
    conditions: list[Condition]
    actions: list[RuleAction]
    source: str  # Standard, guideline, etc.
    effective_date: date
    expiration_date: date | None
    metadata: dict

@dataclass(frozen=True)
class Condition:
    """Condición en una regla."""
    attribute: str
    operator: ConditionOperator
    value: Any
    connector: ConditionConnector = ConditionConnector.AND

class ConditionOperator(Enum):
    EQUALS = "equals"
    NOT_EQUALS = "not_equals"
    GREATER_THAN = "greater_than"
    LESS_THAN = "less_than"
    GREATER_OR_EQUAL = "greater_or_equal"
    LESS_OR_EQUAL = "less_or_equal"
    CONTAINS = "contains"
    IN = "in"
    BETWEEN = "between"
    MATCHES_PATTERN = "matches_pattern"

@dataclass(frozen=True)
class RuleAction:
    """Acción triggered por una regla."""
    action_type: ActionType
    message: str
    priority: Priority
    compliance_status: ComplianceStatus

class ActionType(Enum):
    ALERT = "alert"
    RECOMMENDATION = "recommendation"
    COMPLIANCE_WARNING = "compliance_warning"
    SAFETY_STOP = "safety_stop"
    MAINTENANCE_REQUIRED = "maintenance_required"
```

### Motor de Evaluación

```python
class RulesEngine:
    """Motor de reglas estilo Drools."""
    
    def __init__(self):
        self.repository = RuleRepository()
        self.evaluator = ConditionEvaluator()
    
    async def evaluate(
        self,
        context: dict,
        categories: list[RuleCategory] | None = None,
    ) -> list[RuleMatch]:
        """Evalúa reglas contra un contexto."""
        matches = []
        
        rules = self.repository.get_rules(categories=categories)
        
        for rule in rules:
            if not self._is_effective(rule):
                continue
            
            conditions_met, conditions_not_met = await self._evaluate_conditions(
                rule.conditions, context
            )
            
            matched = len(conditions_not_met) == 0
            
            if matched:
                matches.append(RuleMatch(
                    rule_id=rule.rule_id,
                    rule_name=rule.name,
                    matched=True,
                    conditions_met=[c.attribute for c in conditions_met],
                    conditions_not_met=[],
                    action=rule.actions[0].message if rule.actions else None,
                    priority=rule.priority,
                ))
        
        # Sort by priority
        matches.sort(key=lambda m: m.priority.value, reverse=True)
        return matches
    
    async def _evaluate_conditions(
        self,
        conditions: list[Condition],
        context: dict,
    ) -> tuple[list[Condition], list[Condition]]:
        """Evalúa condiciones de una regla."""
        met = []
        not_met = []
        
        for condition in conditions:
            value = context.get(condition.attribute)
            if value is None:
                not_met.append(condition)
                continue
            
            result = self.evaluator.evaluate(condition, value)
            if result:
                met.append(condition)
            else:
                not_met.append(condition)
        
        return met, not_met
```

- ✅ Compliance checking automatizado

### Negativas
- ⚠️ Requiere mantenimiento de reglas
- ⚠️ Posibles conflictos entre reglas
