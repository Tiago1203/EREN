# ADR-0011: EPIC 11 - Knowledge Governance & Lifecycle Architecture

## Status

**Accepted** - 2026-07-23

## Context

Necesitamos administrar el ciclo de vida completo del conocimiento biomédico:
1. Auditoría completa
2. Versionado y snapshots
3. Políticas de retención
4. Rollback y recuperación
5. Compliance regulatorio

## Decision

### 1. Arquitectura de Lifecycle

```
Lifecycle/
├── KnowledgeSnapshot ─────────────► Snapshot model
├── InMemoryLifecycleManager ───────► Manager
└── RollbackManager ───────────────► Rollback handling
```

**Etapas:**
- DRAFT → REVIEW → PUBLISHED → ARCHIVED/DEPRECATED

### 2. Arquitectura de Compliance

```
Compliance/
├── GovernancePolicy ──────────────► Policy model
├── InMemoryComplianceManager ─────► Manager
└── RetentionEnforcer ─────────────► Retention rules
```

**Políticas:**
- PERMANENT: Nunca eliminar
- STANDARD: 7 años (regulación clínica)
- SHORT_TERM: 2 años
- REVIEW_REQUIRED: Revisión periódica

### 3. Arquitectura de Audit

```
Audit/
├── AuditEntry ───────────────────► Entry model
├── InMemoryAuditManager ──────────► Manager
└── AuditReporter ────────────────► Report generation
```

**Acciones auditadas:**
- CRUD operations
- Publishing/Archiving
- Rollbacks
- Policy changes

## Consequences

### Positive

1. **Auditability**: Cada cambio es auditable
2. **Compliance**: Cumple regulaciones
3. **Recoverability**: Rollback disponible
4. **Governance**: Políticas claras

### Negative

1. **Storage**: Más snapshots
2. **Complexity**: Más componentes

### Neutral

1. **In-memory**: Para demo; production requiere DB
2. **Full lifecycle**: Implementación completa

## Alternatives Considered

### 1. Sin auditoría
- **Rejected**: No hay trazabilidad

### 2. Sin políticas de retención
- **Rejected**: Violación de regulaciones

### 3. Sin rollback
- **Rejected**: No hay recuperación

## Implementation Notes

### Snapshot Creation

```python
snapshot = await lifecycle.create_snapshot(
    document_id=doc_id,
    content=content,
)
# Stores content hash and metadata
```

### Retention Periods

```python
RetentionPolicy = {
    PERMANENT: 100 years,
    STANDARD: 7 years,  # Clinical regulation
    SHORT_TERM: 2 years,
    REVIEW_REQUIRED: 1 year,
}
```

### Audit Actions

```python
AuditAction = [
    CREATED, UPDATED, DELETED,
    PUBLISHED, ARCHIVED, SUPERSEDED,
    VIEWED, EXPORTED, ROLLED_BACK,
    STATUS_CHANGED, POLICY_APPLIED,
]
```
