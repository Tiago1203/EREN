# ADR-0909: Model Versioning

**Status:** Accepted

**Date:** 2026-07-20

**Deciders:** Architecture Board

---

## Context

Necesitamos versionar modelos para tracking y rollback.

---

## Decision

Model Versioning con:

1. **Registry**
   - Model artifacts
   - Training config
   - Evaluation results
   - Metadata

2. **Lifecycle**
   - Development
   - Staging
   - Production
   - Archived

3. **Operations**
   - A/B routing
   - Canary deployments
   - Rollback
   - Shadow mode

---

## Consequences

**Pros:**
- Reproducibility
- Safe deployments
- Audit trail

**Cons:**
- Storage overhead
- Complex routing
