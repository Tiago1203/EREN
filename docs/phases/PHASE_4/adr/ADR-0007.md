# ADR-0007: EPIC 7 - Citation & Traceability Architecture

## Status

**Accepted** - 2026-07-23

## Context

Necesitamos garantizar trazabilidad completa para respuestas clínicas:
1. Generar citas verificables en múltiples estilos
2. Resolver identificadores (DOI, PMID)
3. Verificar fuentes clínicas
4. Mantener pista de auditoría

## Decision

### 1. Arquitectura de Citations

```
Citations/
├── Citation ──────────────► Modelo base
├── ClinicalCitationBuilder ──► Builder clínico
└── CitationValidator ──────► Validador
```

**Estilos soportados:**
- APA (Autor, año, título, revista)
- Vancouver (Numérico)
- MLA, Chicago, AMA

### 2. Arquitectura de References

```
References/
├── Reference ─────────────► Modelo de referencia
├── DOIResolver ───────────► CrossRef API
├── PubMedResolver ─────────► PubMed API
└── ReferenceIndexer ───────► Búsqueda por términos
```

### 3. Arquitectura de Sources

```
Sources/
├── SourceEvidence ──────────► Evidencia de fuente
├── ClinicalSourceVerifier ───► Verificación clínica
├── SourceTracker ────────────► Uso tracking
└── AuditTrail ──────────────► Auditoría
```

**Evidence Levels:**
- LEVEL_1: Systematic review, RCT
- LEVEL_2: RCT, cohort
- LEVEL_3: Case-control
- LEVEL_4: Expert opinion
- LEVEL_5: Animal studies

### 4. Confiabilidad por Tipo

| Source Type | Reliability |
|------------|-------------|
| PUBMED | 0.9 |
| CLINICAL_GUIDELINE | 0.9 |
| PMC | 0.85 |
| REGULATORY | 0.85 |
| TEXTBOOK | 0.7 |
| MANUFACTURER | 0.6 |
| WEBSITE | 0.4 |

## Consequences

### Positive

1. **Traceability**: Cada cita es auditable
2. **Verification**: Fuentes verificadas por tipo
3. **Flexibility**: Múltiples estilos de citación
4. **Compliance**: Auditoría completa

### Negative

1. **Complexity**: Más componentes
2. **Latency**: API calls para resolver DOI/PMID

### Neutral

1. **Caching**: Requerido para performance
2. **Validation**: Puede rechazar citas inválidas

## Alternatives Considered

### 1. Sin citación estructurada
- **Rejected**: Sin trazabilidad

### 2. Sin verificación de fuentes
- **Rejected**: Confiabilidad desconocida

### 3. Sin auditoría
- **Rejected**: No hay compliance

## Implementation Notes

### Citation Formatting

```python
class Citation:
    def format(self) -> str:
        formatters = {
            APA: self._format_apa,
            VANCOUVER: self._format_vancouver,
            ...
        }
        return formatters[self.style]()
```

### Source Verification

```python
async def verify(self, source: SourceEvidence) -> SourceEvidence:
    source.status = SourceStatus.VERIFIED
    source.reliability_score = self._reliability_weights[source.type]
    source.evidence_level = self._determine_level(source)
    return source
```
