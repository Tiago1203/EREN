# ADR-0003: EPIC 3 - Clinical Embedding Architecture

## Status

**Accepted** - 2026-07-23

## Context

Necesitamos generar embeddings especializados para conocimiento biomédico que puedan:
1. Capturar semántica médica
2. Ser cacheados para rendimiento
3. Ser versionados para trazabilidad
4. Ser validados para calidad
5. Procesarse en batch para eficiencia

## Decision

### 1. Arquitectura de Providers

Usamos múltiples providers según el entorno:

```
Providers/
├── MockEmbeddingProvider ─── Testing, fallback
├── SentenceTransformerProvider ─ Local models
├── OllamaProvider ─────────── Local LLM (nomic-embed-text)
└── OpenAIProvider ─────────── Cloud API (text-embedding-3)
```

**Ventajas:**
- Flexibilidad según entorno (dev, staging, prod)
- Mock para testing sin costos
- Ollama para privacidad de datos
- OpenAI para máxima calidad

### 2. Arquitectura de Cache

Cache de dos niveles:

```
Cache/
├── InMemoryEmbeddingCache ─── LRU en memoria
└── PersistentEmbeddingCache ── JSON en disco
```

**Ventajas:**
- L1: Rápido, volatile
- L2: Persistente, más lento
- LRU: Evita overflow de memoria

### 3. Arquitectura de Versioning

Versionado semántico:

```
VersionManager/
├── InMemoryVersionManager
└── VersionComparator
```

**Ventajas:**
- Trazabilidad de cambios
- Comparación de versiones
- Deprecación gradual

## Consequences

### Positive

1. **Flexibilidad**: Múltiples providers según entorno
2. **Rendimiento**: Cache L1 + L2
3. **Trazabilidad**: Versionado de embeddings
4. **Escalabilidad**: Batch processing

### Negative

1. **Complejidad**: Más componentes
2. **Consistencia**: Diferentes providers pueden producir resultados distintos

### Neutral

1. **Latencia**: Ollama más lento que OpenAI
2. **Costo**: OpenAI tiene costo por token

## Alternatives Considered

### 1. Solo un provider (OpenAI)
- **Rejected**: Costo, privacidad de datos

### 2. Sin cache
- **Rejected**: Regeneración costosa

### 3. Sin versionado
- **Rejected**: Sin trazabilidad

## Implementation Notes

### Provider Factory

```python
class EmbeddingProviderFactory:
    @staticmethod
    def create(provider_type: str, **kwargs):
        providers = {
            "mock": MockEmbeddingProvider,
            "sentence-transformers": SentenceTransformerProvider,
            "ollama": OllamaProvider,
            "openai": OpenAIProvider,
        }
        return providers[provider_type](**kwargs)
```
