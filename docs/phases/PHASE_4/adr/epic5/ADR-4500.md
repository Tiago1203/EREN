# ADR-4500: Citation Generation Strategy

## Estado
**Proposed** 📝

## Contexto
Las respuestas clínicas requieren citación precisa para:
- Trazabilidad de fuentes
- Compliance regulatorio
- Accountability médico
- Reproducibilidad de recomendaciones

## Decisión

### Formatos de Citación

| Formato | Uso |
|---------|-----|
| APA | Publicaciones académicas |
| Vancouver | Medicina clínica |
| Numeric | Respuestas cortas |
| Chicago | Documentos largos |

### Tipos de Fuente Citables

| Fuente | Evidence Level | Ejemplo |
|--------|----------------|---------|
| PubMed | 1a-4 | Artículos revisados por pares |
| Clinical Trials | 1b-2b | Estudios clínicos |
| Guidelines | 1a-1b | Guías oficiales |
| Device Manuals | 4-5 | Documentación de fabricante |
| Standards | 4 | ISO, IEC, AAMI |
| Regulations | 5 | FDA, EMA |

### Generación de Citas

```
Retrieved Knowledge
    │
    ├──→ Extract Citation Metadata
    │        ├── Title
    │        ├── Authors
    │        ├── Publication
    │        ├── Year
    │        └── DOI/URL
    │
    ├──→ Assign Evidence Level
    │        └── Based on Source Type
    │
    ├──→ Format Citation
    │        └── APA/Vancouver/Numeric
    │
    └──→ Inject into Response
             └── Inline + Reference List
```

### Trazabilidad de Evidencia

Cada claim en la respuesta se mapea a:
- Citation IDs
- Evidence Levels
- Reasoning Path

## Consecuencias

### Positivas
- Citas formateadas profesionalmente
- Trazabilidad completa de fuentes
- Evidence levels para calibrar confianza

### Negativas
- Requiere metadata estructurada
- Más tokens en respuesta final
